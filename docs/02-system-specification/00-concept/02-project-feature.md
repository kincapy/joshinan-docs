# プロジェクト機能

## 概要

日本語学校の業務には「やることが決まっている定型業務」が多い。
しかし、それらは1日で終わらず、途中でデータが追加されたり、複数の作業が並行したりするため、進行管理が煩雑になる。

プロジェクト機能は、**システムが業務の進行を主導し、人間は求められた入力をするだけで作業が完了する**、という世界観を実現する機能。

## システムコンセプトとの関係

[システムコンセプト](./index.md) の設計思想「業務管理をシステムが主導する」を、具体的に実現する仕組み。

| コンセプトの設計思想 | プロジェクト機能での実現 |
|---|---|
| 人間の入力を最小化する | ワークフローに沿って必要な入力だけを順番に求める |
| 業務管理をシステムが主導 | システムがスキルを提示し、進行をガイドする |
| スケジュールベース業務 | プロジェクトとして長期的な進行管理ができる |

## チャットボット機能との違い

どちらもチャットUIだが、目的と動き方が異なる。

| | チャットボット | プロジェクト機能 |
|---|---|---|
| **起点** | 人間が自由に話しかける | システムがスキルを提示する |
| **操作** | 自由な対話。既存データの問い合わせ・修正 | 定義されたワークフローに沿って進行 |
| **データ操作** | 既存データの読み書き・修正 | 新しいデータの追加が中心 |
| **例** | 「この学生を退学にして」「先月の未納者出して」 | 「2026年4月募集」で学生情報を順番に入力→書類生成 |

## 構成要素

### スキル（テンプレート）

業務パターンを事前定義したもの。各スキルには以下が含まれる。

- **目的**: 何を達成するか
- **ゴール**: 完了条件（最初から定義されている）
- **ワークフロー**: どういう順番で何をするか
- **必要な入力**: 人間に求める情報

スキルの例:

| スキル | 目的 | ゴール |
|---|---|---|
| 募集活動 | 新入学期の学生を募集・申請する | 入管への申請書類を提出する |
| ビザ更新 | 在留期間更新の手続き | 更新申請書類を提出する |
| 特定技能申請 | 特定技能の在留資格申請 | 申請書類を提出する |

### プロジェクト（スキルのインスタンス）

スキルを使って開始された、具体的な業務の単位。

- 例: 「2026〜27年4月の募集活動」
- 1つのスキルから複数のプロジェクトが作られる（年度ごと、時期ごと）
- 1日では終わらない。データが来るたびに追加入力し、進行していく

### ワークフロー（スキル内の処理の流れ）

スキルの中で定義された、一連の作業手順。

**例: 募集活動スキルのワークフロー**

```
1. プロジェクト開始
   └ 対象入学期を選択（例: 2026年4月期）

2. 学生情報の入力（繰り返し）
   ├ 申請予定の学生情報を入力
   ├ → 学生ごとに台帳を作成
   ├ → 書類フォルダを作成（Box）
   └ → 入管用書式を生成

3. 書類の準備・確認
   ├ 必要書類の回収状況を管理
   └ 不足書類のアラート

4. 申請書類の出力
   └ 全学生分の申請書類セットを生成

5. 完了
   └ 入管への提出をもって完了
```

## 設計思想

### システム起点

人間が「プロジェクトを作って」と依頼するのではなく、**システムが「これができますよ」と提示する**。
人間はシステムが提示したスキルの中から選んで開始する。

### ガイド型の進行

プロジェクト開始後、システムが順番に「次はこの情報を入れてください」とガイドする。
人間は指示に従って入力するだけで、必要な出力物が生成される。

### 機能追加 = スキルの追加

システムに新しい業務パターンを追加するときは、新しいスキルを定義する。
通常のシステム開発における「機能リリース」は、このスキルのコンポーネントが追加されていくイメージ。

## UIイメージ

- **左サイドバー**: 進行中のプロジェクト一覧
- **メイン領域**: チャットベースのUI（Claudeのチャットのような対話型）
- スキルの選択、入力の要求、進捗の表示がチャット形式で進む

## 今後の要件定義で決めること

- プロジェクトの起案者・作成日時の管理
- 複数人での協働（プロジェクトへの参加・権限）
- スキルの具体的な定義フォーマット
- プロジェクトのステータス管理（進行中・完了・中断など）
- データモデル（Project, Skill, WorkflowStep 等のエンティティ定義）
