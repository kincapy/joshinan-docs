# プロジェクト機能 — UIUX定義

## 画面一覧

| # | 画面 | パス | 説明 |
|---|------|------|------|
| 1 | プロジェクト一覧 | `/project` | 進行中・完了済みプロジェクトの一覧 |
| 2 | プロジェクト詳細 | `/project/[id]` | タスクボード + チャット |
| 3 | スキル管理 | `/project/skills` | スキルの一覧・作成・修正（管理者以上） |

### チャットでのタスク操作

タスクボードのクリック操作に加えて、チャットでも操作できる。

```
User: 「COL-001の健康診断、予約取れたのでステータスを対応中にして」
  ↓
AI: 「COL-001 健康診断個人票のステータスを「対応中」に変更しました。」

User: 「登記事項証明書が届いたからアップロードする」
  ↓
AI: 「ファイルをアップロードしてください。」
  ↓
User: [ファイルをドロップ]
  ↓
AI: 「COL-012 登記事項証明書にファイルを添付し、ステータスを「完了」に変更しました。」
```

```
  ↓

│                                      │

│ 鈴木（閲覧者）              [変更▼]  │
│                                      │
│ [+ メンバーを追加]                    │
│                                      │
│  [閉じる]                            │
└──────────────────────────────────────┘
```

### 全タスク完了 → 成果物生成

```
最後のタスクが「完了」になる
  ↓
AI: 「全ての必須タスクが完了しました。
     申請書類セットを生成しますか？」
     [生成する]  [まだ待つ]
  ↓
User: [生成する]
  ↓
AI: 「申請書類セットを生成しました。」
    ┌──────────────────────────────────┐
    │ 申請書類セット（13ファイル）       │
    │ DOC-001 申請書（申請人用）.pdf     │
    │ DOC-002 申請書（所属機関用）.pdf   │
    │ DOC-003 雇用契約書.pdf            │
    │ ...                              │
    └──────────────────────────────────┘
    [一括ダウンロード（ZIP）]
    [Boxに保存]
```

## スキル管理画面

管理者以上がアクセスできるスキルの管理画面。プロンプトベースで操作する。

```
┌──────────────────────────────────────────────────────┐
│  スキル管理                                          │
├──────────────────────────────────────────────────────┤
│                                                      │
│  ● 有効 (1)  ○ 無効  ○ 全て                         │
│                                                      │
│  ┌────────────────────────────────────────────────┐  │
│  │ 特定技能申請                                    │  │
│  │ 目的: 特定技能の在留資格変更許可申請             │  │
│  │ タスク数: 36  バージョン: 1                     │  │
│  │                          [詳細] [修正]          │  │
│  └────────────────────────────────────────────────┘  │
│                                                      │
│  ┌─────────────────────────────────────────────┐     │
│  │ プロンプトでスキルを作成・修正       [送信]  │     │
│  └─────────────────────────────────────────────┘     │
│                                                      │
└──────────────────────────────────────────────────────┘
```

### スキル修正の例

```
User: 「特定技能申請スキルに、2025年4月新設の協力確認書（DOC-013）を
       必須タスクとして追加して」
  ↓
AI: 「以下の変更をスキル「特定技能申請」に適用します。」
    ┌──────────────────────────────────────┐
    │ 追加タスク: DOC-013 協力確認書        │
    │ カテゴリ: 書類作成                    │
    │ アクション: 自動生成                  │
    │ 必須: はい                           │
    │                                      │
    │  [決裁申請]  [キャンセル]             │
    └──────────────────────────────────────┘
```
