# 経理・会計

常南国際学院（株式会社スイッチサービス）の経理・会計業務に関するドメインナレッジ。
学費管理（06-tuition）やエージェント管理（07-agent-management）で扱う個別の取引データを、**会社全体の複式簿記**として仕訳・集計するための業務知識をまとめる。

## 担当者

| 担当者 | 役割 |
|--------|------|
| 笹目（経理責任者） | 元帳管理、仕訳ルール策定、決算処理 |
| Gotoさん | 入金データ入力、月次報告 |
| 兄貴 | 経営判断、最終確認 |

## スコープ

| 対象 | 具体的な内容 |
|------|-------------|
| 勘定科目体系 | 全科目・補助科目の一覧と税区分 |
| 仕訳ルール | 売上計上・入金消込・費用計上・源泉徴収の仕訳パターン |
| 仕訳帳フォーマット | 元帳の列構成・伝票番号ルール・摘要の書き方 |
| 消費税 | 税抜方式、品目ごとの課税/非課税区分 |
| 決算ルール | 会計年度（4月〜3月）、月次締め、年度末決算仕訳 |
