# 業務フロー — 月次経理サイクル

## フロー図

```mermaid
graph LR
    A[月次請求確定] --> B[売上仕訳]
    B --> C[入金処理]
    C --> D[売掛金消込]
    D --> E[費用仕訳]
    E --> F[通帳突合]
    F --> G[仕訳帳出力]
    G --> H[月次締め]
```

## 会計方式

| 項目 | 採用方式 |
|------|---------|
| 認識基準 | **発生主義**（請求時に売上計上、入金時に売掛金消込） |
| 消費税 | **税抜方式**（仮払消費税・仮受消費税を個別仕訳） |
| 会計年度 | **4月1日〜3月31日** |
| 月次締め | **月末締め**（売上計上は月末日付で処理） |

## 各ステップの詳細

### 1. 月次請求確定

- **担当:** Gotoさん（学費請求の生成）
- **タイミング:** 月初に前月分の請求を確定
- **内容:**
  - 学生ごとの請求データを確定（品目×月の組み合わせ）
  - 3月・8月は学費を除外（寮費のみ）
  - 特定技能の月額支援費も同タイミングで生成
- **入力:** 学生マスタ、品目マスタ、月別請求テーブル
- **出力:** 確定済みの請求レコード

### 2. 売上仕訳

- **担当:** 笹目
- **タイミング:** 月末日付で計上
- **内容:**
  - 確定した請求を「売掛金 / 売上高」で仕訳
  - 課税品目の場合は仮受消費税を別建てで計上
  - 摘要には学生名＋品目＋対象月を記載

**仕訳パターン（非課税品目 — 学費・寮費等）:**
```
借方: 売掛金        94,300
    貸方: 学費売上   69,300
    貸方: 寮費売上   25,000
```

**仕訳パターン（課税品目 — 教材費等）:**
```
借方: 売掛金        33,000
    貸方: 教材費売上  30,000
    貸方: 仮受消費税   3,000
```

### 3. 入金処理

- **担当:** Gotoさん
- **タイミング:** 入金の都度
- **内容:**
  - 現金入金: 学生が事務所に持参 → 「現金 / 売掛金」で仕訳
  - 銀行振込: 通帳CSVから照合 → 「普通預金 / 売掛金」で仕訳
- **入力:** 現金受領、銀行CSV（足利銀行・りそな銀行）
- **出力:** 入金レコード、仕訳データ

### 4. 売掛金消込

- **担当:** 自動（入金処理と連動）
- **内容:**
  - 入金額を未消込の請求に古い月から順に充当
  - 完全にカバーした請求は「完了」ステータスに遷移
  - 過払い（前払い）が発生した場合は前受金として処理

**過払い時の仕訳:**
```
借方: 普通預金       120,000
    貸方: 売掛金      94,300
    貸方: 前受金      25,700
```

### 5. 費用仕訳

- **担当:** 笹目
- **タイミング:** 請求書受領時・支払時
- **内容:**
  - エージェント手数料 → 「業務委託料」として一括計上
  - 寮の家賃 → 「地代家賃」として月次計上
  - 水光熱費 → 「水道光熱費」として月次計上
  - 人件費 → 「給料手当」「役員報酬」「法定福利費」等
  - その他費用 → 通信費、車両費、支払手数料等

**エージェント手数料の仕訳（請求時）:**
```
借方: 業務委託料     200,000
借方: 仮払消費税      20,000
    貸方: 未払金（業務委託料） 220,000
```

**エージェント手数料の仕訳（支払時）:**
```
借方: 未払金（業務委託料）   220,000
    貸方: 普通預金            199,580
    貸方: 預り金（業務委託料）  20,420  ← 源泉徴収分
```

### 6. 通帳突合

- **担当:** 笹目
- **タイミング:** 月次締め前
- **内容:**
  - 銀行通帳（三井住友銀行日本橋東支店 普通7831007）の入出金と仕訳帳を照合
  - 不一致があれば原因を特定して修正仕訳を起票
- **入力:** 通帳CSV、仕訳帳
- **出力:** 突合完了した仕訳帳

### 7. 仕訳帳出力

- **担当:** 笹目
- **タイミング:** 月次締め後
- **内容:**
  - 月次の全仕訳を元帳フォーマットで出力
  - Excel形式が基本（CSV補助出力あり）
- **出力:** 月次仕訳帳（Excel）

### 8. 月次締め

- **担当:** 笹目
- **タイミング:** 月末
- **内容:**
  - 当月の仕訳がすべて完了していることを確認
  - 残高試算表で借方合計＝貸方合計を確認
  - 翌月への繰越処理

## 年度末の追加処理

年度末（3月末）には、以下の決算仕訳が追加で必要になる。

| 決算仕訳 | 内容 |
|----------|------|
| 消費税の相殺 | 仮払消費税と仮受消費税を相殺し、差額を未払消費税に振替 |
| 更新料の期間按分 | 繰延資産（事務所更新料等）の月割償却 |
| 未払費用の計上 | 役員報酬・給与の未払分を計上 |
| 法人税等の計上 | 法人税・住民税・事業税の計上 |

::: info 繰越欠損金
第7期末時点で繰越欠損金が約3,913.5万円あるため、当面は法人税がかからない見込み。
:::
