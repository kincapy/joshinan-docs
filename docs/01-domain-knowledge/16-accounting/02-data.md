## 品目ごとの消費税区分

常南国際学院は**株式会社立の日本語学校**であり、学校法人ではない。非課税の適用範囲は税理士確認が必要。

| 品目 | 税区分 | 根拠 |
|------|--------|------|
| 学費（授業料） | **非課税** | 学校教育法に準ずる教育役務の提供（消費税法別表第一第十一号） |
| 入学金 | **非課税** | 同上 |
| 教材費 | **課税（10%）** | 物品の販売に該当 |
| 施設費 | **非課税** | 授業料に準ずるもの（要税理士確認） |
| 課外活動費 | **課税（10%）** | 教育の範疇を超えるイベント費用 |
| 検定料 | **非課税** | 国の法令に基づく検定試験（消費税法別表第一第十二号） |
| 保険料 | **非課税** | 保険の対価は非課税 |
| 寮費 | **非課税** | 住宅の貸付け（消費税法別表第一第十三号） |
| 寮初期費用 | **非課税** | 住宅貸付けに準ずる |
| その他 | **個別判断** | 内容による |
| 特定技能紹介料 | **課税（10%）** | 人材紹介サービス |
| 特定技能支援費 | **課税（10%）** | 支援業務サービス |

::: warning 株式会社立の注意
スイッチサービスの元帳では売上に仮受消費税が発生していたため、課税売上として処理していた実績がある。非課税適用の可否は税理士に確認が必要。
:::

## 仕訳帳フォーマット

### 列構成

元帳（マザーデータシート）のフォーマット。システムの仕訳帳出力もこの形式に合わせる。

| 列名 | 型 | 説明 |
|------|-----|------|
| 取引No | 数値 | 伝票番号（通し連番） |
| 取引日 | 日付 | 仕訳日（YYYY-MM-DD） |
| 会計年度 | 文字列 | 西暦年度（例: 2024） |
| 令和年度 | 文字列 | 和暦年度（例: R6） |
| 勘定科目 | 文字列 | 勘定科目名 |
| 補助科目 | 文字列 | 補助科目名（該当なしの場合は空欄） |
| 摘要 | 文字列 | 取引の内容説明 |
| 借方金額 | 数値 | 借方金額（該当なしの場合は0） |
| 貸方金額 | 数値 | 貸方金額（該当なしの場合は0） |
| 残高 | 数値 | 科目ごとの残高 |

### 複合仕訳の表現

同じ取引Noで複数行を使い、複借複貸（複合仕訳）を表現する。1仕訳の借方合計＝貸方合計になること。

**実データの例（R6期）:**

| 取引No | 取引日 | 勘定科目 | 補助科目 | 摘要 | 借方 | 貸方 |
|--------|--------|----------|---------|------|------|------|
| 9316 | 2024-04-01 | 通信費 | — | GOOGLE*GSUITE COMMON | 2,184 | 0 |
| 9316 | 2024-04-01 | 仮払消費税 | — | GOOGLE*GSUITE COMMON | 218 | 0 |
| 9316 | 2024-04-01 | 未払金 | 三井住友カード… | GOOGLE*GSUITE COMMON | 0 | 2,402 |

### 伝票番号ルール

- 全期間通しの連番（R2期 No.2984〜 → R6期 No.9316〜）
- システムでは期首リセットの連番でも可

### 摘要の記載パターン

| パターン | 例 | 用途 |
|----------|-----|------|
| 請求書番号＋相手先＋月 | No.374 株式会社サンクリスタ－・4月ご請求について | 売上計上時 |
| 振替＋相手先カナ | 振替 カ）サンクリスタ- | 入金（銀行振込）時 |
| 学生名＋品目＋月 | 2410001 ファム・ティ・ハ / 学費 / 2025年4月分 | 学生向け請求の仕訳時 |
| サービス名 | GOOGLE*GSUITE COMMON | クレカ経費 |
| 相手先名＋内容 | W21 リユウザキ ユウジ | 給与・立替精算 |

## 元帳のカテゴリ分類

元帳にはマッピング定義シートがあり、以下のルールで分類している。

| 分類対象 | 分類方法 | カテゴリ例 |
|----------|---------|-----------|
| 売上 | 相手先→カテゴリー | スイッチ関連、エイジ関連、常南交通 |
| 販管費 | 摘要キーワード→フラグ | 常南立替、GCP、ウズベキスタン事業等 |
