# 特定技能・職業紹介 データ定義

## 案件 (Case)

特定技能の申請案件を管理する中心的なデータ。学生と企業を紐づけ、案件全体の進捗を追跡する。

| 項目 | 内容 | 備考 |
|------|------|------|
| 案件ID | 案件を一意に識別 | |
| 学生 | 対象の学生 | 学生マスタへの参照 |
| 企業 | 受入れ企業 | 企業マスタへの参照 |
| 分野 | 特定技能の分野 | 下記「対象分野」参照 |
| ステータス | 案件の進捗（8段階） | 下記「ステータス遷移」参照 |
| 申請日 | 入管への申請日 | |
| 許可日 | 在留資格変更の許可日 | |
| 入社日 | 受入れ企業への入社日 | |
| 紹介料 | 職業紹介手数料 | デフォルト 150,000円 |
| 月額支援費 | 登録支援費 | デフォルト 10,000円 |

### ステータス遷移

```
営業中 → 書類準備中 → 申請中 → 審査中 → 許可済み → 入社済み → 支援中 → 終了
```

| ステータス | 意味 |
|---|---|
| 営業中 | 企業と候補者のマッチング段階 |
| 書類準備中 | 内定後、申請書類を準備中 |
| 申請中 | 入管に申請書類を提出済み |
| 審査中 | 入管が審査中 |
| 許可済み | 在留資格変更が許可された |
| 入社済み | 受入れ企業に入社した |
| 支援中 | 義務的支援を実施中 |
| 終了 | 退職/帰国/期間満了等で支援終了 |

### 対象分野（当社実績のある5分野）

| 分野 | 備考 |
|------|------|
| 介護 | |
| 宿泊 | |
| 外食業 | |
| 飲食料品製造業 | |
| 自動車運送業 | |

::: info 制度上の全分野（2025年4月時点：16分野）
上記5分野以外にも、ビルクリーニング、素形材・産業機械・電気電子情報関連製造業、建設、造船・舶用工業、自動車整備、航空、農業、漁業など全16分野が存在する。
:::

### 自動入力のデータソース

申請書類のフィールドは以下の3つのソースから自動入力できる。

| ソース | 内容 |
|---|---|
| 学生マスタ | 氏名、国籍、生年月日、パスポート、在留情報、住所、試験合格情報 |
| 企業マスタ | 企業名、代表者、住所、雇用条件 |
| 登録支援機関固定情報 | 常南交通の機関名・登録番号・住所等（全書類で共通） |

## 書類ステータス

各書類の進捗は以下のステータスで管理する。

| ステータス | 意味 |
|---|---|
| 未着手 | まだ手をつけていない |
| 作成中 | 当社が作成中（DOC系） |
| 収集依頼中 | 申請人または企業に依頼済み（COL系） |
| 収集済み | 受領済み |
| 自動入力済み | 学生DBからの自動入力が完了 |
| 確認待ち | 内容確認待ち |
| 完了 | 最終確認済み、申請可能な状態 |
| 不要 | この案件では不要（免除・条件外） |
| 差戻し | 不備があり再提出が必要 |

## 条件分岐ルール

書類の必要/不要は、学生の属性や分野によって変わる。

### 技能実習からの移行

技能実習2号を修了した者は、技能試験（COL-003）と日本語試験（COL-004）が**免除**される。

### 分野別

| 条件 | 追加で必要な書類 |
|---|---|
| 介護 | COL-020（介護日本語評価試験合格証） |
| 宿泊 | COL-021（旅館業許可証の写し） |
| 自動車運送業 | COL-022（運転免許証）、COL-023（協議会構成員資格証明書） |

### 国籍別

| 条件 | 追加で必要な書類 |
|---|---|
| カンボジア/タイ/ベトナム国籍 | COL-019（二国間取決に係る書類） |

### 保険種別

| 条件 | 追加で必要な書類 |
|---|---|
| 国民健康保険加入者 | COL-008、COL-009 |
| 国民年金加入者 | COL-010、COL-011 |

## データ基盤の共通化について

特定技能の業務で必要なデータの多くは、学生管理（02-student-management）で既に保持しているデータと重複する。新たに独自のデータとして管理するのは以下の通り。

| データ | 新規/共有 | 備考 |
|--------|-----------|------|
| 案件 | **新規** | 特定技能固有のデータ |
| 受入れ企業 | **新規** | 特定技能固有のデータ |
| 申請書類・証憑書類 | **新規** | 書類ステータス管理 |
| 学生基本情報 | 共有 | 02-student-management で管理 |
| パスポート・在留情報 | 共有 | 02-student-management で管理 |
| 日本語レベル | 共有 | 02-student-management で管理 |
| エージェント情報 | 共有 | 07-agent-management で管理 |
