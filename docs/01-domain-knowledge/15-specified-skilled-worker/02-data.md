# 特定技能・職業紹介 データ定義

## 求人情報 (JobPosting)

企業からの求人情報を管理する。特定技能の分野ごとに分類。

| 項目 | 内容 | 備考 |
|------|------|------|
| 企業名 | 求人を出す企業 | |
| 職種分野 | 特定技能の12分野から選択 | 下記参照 |
| 職種詳細 | 具体的な職務内容 | |
| 勤務地 | 勤務先の所在地 | |
| 給与（月額） | 月額給与 | 円 |
| 雇用形態 | 正社員・契約社員等 | |
| 求人ステータス | 募集中/募集終了/充足 | |
| 募集人数 | 何名募集するか | デフォルト1名 |
| 備考 | 自由記述 | |

## マッチング (JobMatch)

学生と求人のマッチング状況を管理する。

| 項目 | 内容 | 備考 |
|------|------|------|
| 求人 | どの求人に対するマッチングか | 求人情報への参照 |
| 学生 | どの学生のマッチングか | 学生管理への参照 |
| マッチングステータス | 候補/推薦中/面接中/内定/辞退/不合格 | |
| 推薦日 | 企業に推薦した日 | |
| 面接日 | 面接実施日 | |
| 内定日 | 内定が出た日 | |
| 備考 | 自由記述 | |

::: info マッチングステータスの遷移
候補 → 推薦中 → 面接中 → 内定 or 不合格 or 辞退
各段階で日付を記録することで、マッチングの進捗を追跡できる。
:::

## 支援計画 (SupportPlan)

特定技能外国人の支援計画を管理する。登録支援機関としての業務に対応。

| 項目 | 内容 | 備考 |
|------|------|------|
| 学生 | 支援対象の学生 | 学生管理への参照 |
| 支援開始日 | 支援の開始日 | |
| 支援終了日 | 支援の終了日 | 継続中の場合は空 |
| 登録支援機関名 | 支援を行う機関の名称 | |
| 月額支援料 | 月額の支援費用 | 円 |
| 支援内容 | 支援の具体的な内容 | |
| ステータス | 実施中/完了/取消 | |

## データ基盤の共通化について

特定技能の業務で必要なデータの多くは、学生管理（02-student-management）で既に保持しているデータと重複する。新たに独自のデータとして管理するのは以下の3つ。

| データ | 新規/共有 | 備考 |
|--------|-----------|------|
| 求人情報 | **新規** | 特定技能固有のデータ |
| マッチング | **新規** | 学生IDで学生管理と連携 |
| 支援計画 | **新規** | 学生IDで学生管理と連携 |
| 学生基本情報 | 共有 | 02-student-management で管理 |
| 日本語レベル | 共有 | 02-student-management で管理 |
| 在留資格情報 | 共有 | 02-student-management で管理 |
| エージェント情報 | 共有 | 07-agent-management で管理 |
