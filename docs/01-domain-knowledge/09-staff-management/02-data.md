# 教職員管理 データ定義

## 教職員 (Staff)

教職員の基本情報・雇用形態・資格・給与形態を管理するデータオブジェクト。

| 項目 | 内容 | 備考 |
|------|------|------|
| 氏名 | フルネーム | |
| メールアドレス | 連絡先メール | 任意 |
| 電話番号 | 連絡先電話番号 | 任意 |
| 雇用形態 | 常勤 / 非常勤 / パート | |
| 役職 | 校長 / 教務主任 / 常勤講師 / 非常勤講師 / 生活指導員 / 事務担当 / 経営者 | |
| 教員資格種別 | 取得済み資格の種類（複数可） | 教員のみ |
| 資格取得日 | 各資格の取得年月日 | |
| 入社日 | 入社年月日 | |
| 退職日 | 退職年月日 | 在職中は空 |
| 担当クラス | 現在担当しているクラス | 時間割から参照 |
| 週間コマ数上限 | 週間担当コマ数の上限値 | 常勤: 25単位時間以内（望ましくは16コマ）。実績値は時間割から集計 |
| 給与形態 | 月給 / 時給 / コマ給 | 非常勤の場合に重要 |

::: info 給与形態の違い
- **コマ給方式**: 1コマ2,000円。授業以外の残務に報酬が支払われないケースがある
- **時給+授業手当方式**: 時給1,200円 + 授業手当800円。残務も時給で支払われる
:::

## 教員資格 (TeacherQualification)

教員が保有する資格情報。1人の教員が複数の資格を持つことがある。

| 項目 | 内容 | 備考 |
|------|------|------|
| 教職員ID | どの教職員の資格か | Staff への紐づけ |
| 資格種別 | 主専攻 / 副専攻 / 420時間 / 検定合格 / 登録日本語教師 | いずれか + 大卒以上 |
| 取得日 | 資格を取得した日 | |
| 有効期限 | 資格の有効期限 | 登録日本語教師の場合に該当 |
| 備考 | 資格に関する補足情報 | 任意 |

### 資格種別の詳細

| 資格種別 | 説明 |
|---------|------|
| 主専攻 | 日本語教育を主専攻として大学・大学院を卒業 |
| 副専攻 | 日本語教育を副専攻として大学・大学院を卒業 |
| 420時間 | 日本語教師養成講座（420時間以上）修了。通常3〜6ヶ月で修了可能 |
| 検定合格 | 日本語教育能力検定試験合格 |
| 登録日本語教師 | 国家資格。現在移行中の新制度 |
