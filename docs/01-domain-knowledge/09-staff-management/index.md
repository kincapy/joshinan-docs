# 教職員管理

教員資格・配置の管理、日本語教師の業務・特性、事務職・生活指導員の役割を記載するカテゴリです。

## 組織体制

### ロール（役割）の大分類

日本語学校の運営に関わるロールは、大きく「経営」「事務サイド」「教員サイド」の3つに分かれる。

| 大分類 | 代表的なロール | 主な責務 |
|--------|--------------|---------|
| 経営 | 経営者 | 学校全体の運営・会議、コンプライアンス対応、対外折衝 |
| 事務サイド | 校長 / 事務局長、生活指導員、事務担当者 | ビザ更新統括、募集活動統括、学生の生活支援、人事・総務・経理 |
| 教員サイド | 教務主任、常勤講師、非常勤講師 | 教育課程編成、授業・教務事務、学生の進捗管理 |

::: info 校長 / 事務局長の位置づけ
事務サイド全体を統括するロール。ビザ更新周りの統括、募集活動の統括など、いわゆる「ジョブディスクリプション」的な位置づけで業務全般を管轄する。
:::

### 基本人員構成

| 役職 | 役割 | 必須数 |
|------|------|--------|
| 校長 | 機関全体の統括・最終判断者 | 1名 |
| 主任教員（教務主任） | 教育課程の編成・教員指導・授業の質管理 | 1名 |
| 常勤日本語講師 | 授業・教務事務・学生対応の中核人材 | 最低3名以上 |
| 非常勤日本語講師 | 授業担当 | 必要数 |
| 生活指導員 | 学生の生活・健康・進路支援 | 1名以上 |
| 事務担当者 | 人事・総務・経理 | 必要数 |

**最小構成**: 校長1名 + 常勤講師3名 + 事務担当 = 5名体制

### 人的要件

| 役職 | 要件 | 配置数 | 根拠 |
|------|------|--------|------|
| 校長 | 教育に関する識見を有する者 | 1名必須 | 告示基準第7号 |
| 主任教員 | 常勤で日本語教育に関し**3年以上**の経験 | 1名必須 | 告示基準第9号 |
| 教員全体 | 学生**20名につき1名以上** | — | 告示基準第10号 |
| 専任教員 | 学生**40名につき1名以上**。教員の過半数は常勤 | 最低3名以上 | 告示基準第11号 |
| 生活指導担当者 | 生活指導の適当な体制を備えること | 1名以上 | 告示基準第16号・第17号 |
| 事務統括職員 | 事務を統括する者 | 1名 | 告示基準第15号 |

## 日本語教師とは

日本語教師とは、留学生に対して日本語の授業を実施する職種。ただし日本語学校の本質的役割は語学教育にとどまらず、留学生の日本社会への順応支援も含まれるため、業務は非常に広範囲。

### 教員の資格要件（いずれか + 大卒以上）

- 日本語教育を主専攻/副専攻として大学・大学院を卒業
- 日本語教師養成講座（420時間以上）修了（通常3〜6ヶ月で修了可能）
- 日本語教育能力検定試験合格
- ※ 新制度では「登録日本語教師」（国家資格）への移行中

### 授業コマ数の上限

| 役職 | 法定上限 | 望ましい水準 | 根拠 |
|------|----------|------------|------|
| 常勤講師 | 週**25単位時間**以内 | 週16コマ | 告示基準第14号 |
| 主任教員 | 週**25単位時間**以内 | — | 告示基準第14号 |
| 非常勤講師 | 制限なし | — | — |

::: info コマとは
日本語学校の授業は半日制（午前or午後）で、1日4コマ、週20コマで1クラスのカリキュラムが構成される。常勤講師1名で1クラス（週20コマ）をすべて担当することは可能だが、業務負荷を考慮すると週16コマ程度が望ましい。
:::

## 日本語教師の業務

### 1. 授業

- 週16コマ担当の場合、週4日が授業日。1日の授業時間は約3.5時間
- 未経験者にとって3.5時間の授業は体力的に非常に負担が大きい
- 授業後には翌日の担当講師のための「授業記録」を作成する必要あり

### 2. 授業準備

- カリキュラムや教科書・教材の確認が必要
- パワーポイント等が用意されていてもそのままでは使いにくく、加工作業が発生
- 外から見ると簡単そうだが実際には時間がかかる。多くの常勤講師は退勤後に自宅で準備

::: warning 配慮が必要
日本語教師は真面目で職業倫理を重んじる人が多い。非教師が「そこまでやらなくてもいいのでは」と無理解な発言をすると、教師のプライドを傷つけるケースがある。
:::

### 3. 教案・カリキュラムの作成

- 担任クラスのカリキュラムを作成し、生徒の進捗・学習レベルに応じて月単位で調整
- 例: N4合格を目標とするクラスで漢字力不足の場合、翌月の授業に漢字を多く取り入れる
- 初級レベルの授業でも教師は細やかな調整を重ねており、軽んじるべきではない

### 4. 教務事務

- 出席率の入力、宿題回収率の集計、小テストの採点、教材のスキャン
- 作文など専門性の高い業務は日本語教師が対応する必要がある

::: tip 非常勤講師の給与形態
- **コマ給方式**: 例：1コマ2,000円 → 残務に報酬が支払われないケースがある
- **時給+授業手当方式**: 例：時給1,200円 + 授業手当800円 → 残務も時給で支払われる
:::

### 5. 非常勤講師との連絡

常勤講師は、担当クラスの非常勤講師と連絡を取り合い、学生の進捗状況を共有・調整する。

### 6. 生徒との面談

学生の生活・学習・進路について定期的に面談を実施。以下は実際の面談記録の一例:

::: details 面談記録の例
**生活面**
- 引っ越し後は順調。他の国の学生と同居し、日本語・英語で交流
- 風邪気味で薬を飲んでいるが咳が続く
- ビザ更新の書類準備中

**学習面**
- 7月のJLPT N3を受験予定。前回は聴解のみ基準点未達で不合格。今回は自信あり
- 自宅学習は2時間。文法と聴解が中心
- クラス移動後、聴解力が向上し各国の学生との交流で視野が広がった

**アルバイト**
- スーパー: 週4回、14-18時、時給1,005円
- 物流倉庫: 月1-2回、17-22時、時給1,123円
- スーパーの仕事が好き。先輩から真面目と評価されている

**進路**（大卒の場合）
- 特定技能試験（飲食製造業）合格済み
- スーパーで就職希望。茨城・つくば市で働きたい
- 技人国ビザは大学の専攻が特殊すぎて困難と思われる
:::

### 7. 進路指導

- 進学希望者: 専門学校の情報収集、エントリーシート作成支援、面接対策、出願手続きサポート
- 留学生は自立していないため、きめ細かな対応が必要
- **秋以降は担当講師の負担が大きくなるため、後期は授業コマ数を調整するなどの配慮が必要**

### 8. 学生対応

- 生活や進路に不安があると、まず担任教師に相談が来る
- 退勤後や休日でもLINE等で頻繁に連絡あり
- 若くして異国に一人で来ているため、精神的ストレスを抱えている学生が多い
- **常勤講師の見えない精神的負担が発生していることを運営者は十分に理解する必要がある**

## 日本語教師の性格的傾向

::: warning 注意
本内容には主観的な見解が含まれるが、日本語学校を運営する上で理解すべき重要な特徴。
:::

### 職業特性

- 「大卒であれば誰でもなれる」職種。420時間養成講座修了でセカンドキャリアとして就く人も多く、年齢層は高め
- 若年層の日本語教師は非常に稀（国立大学の日本語教育学科でも学部50-60名中、新卒で教師になるのは1名程度）
- 「先生」と呼ばれるため、一方的に価値観を押し付けたり感情的な対応をする者も散見される
- 独特な生態系を持ち、一般企業の常識が通じにくい場面がある

### 学生に寄り添いすぎる傾向

- 学生と非常に距離が近くなるため、留学生の立場で物事を捉える傾向がある
- 例: 学生が軽い怪我をした場合、教師は「かわいそうだから病院に付き添う」と判断するが、運営側は人件費の観点から本人一人で行かせるべきと判断する
- 学生に寄り添う姿勢は重要だが、学校全体の運営を維持するためには合理的な視点も必要
- **社会人経験のある教師は「寄り添い」と「運営」の両立ができる傾向。社会経験が浅い場合は学生寄りに偏りがち**

### 運営側への示唆

- 呼称は「先生」よりも「○○さん」の方が適切
- 日本語教育現場の特殊性を理解しない人物が運営者になることのリスクは大きい
- 不信感の蓄積により「幸せでない職場」になる危険性がある
- **理想的には校長も日本語教師出身であることが望ましい**

::: tip 重要な認識
日本語教師は日本語学校の「主役」。日本語が通じない、必ずしも強い目的意識があるとは限らない留学生を相手に、教育・生活支援を提供する非常に根気がいる仕事。求められる資質は「根気強さ」と「共感性」。
:::

## 校長・事務局長の役割

### 校長は必要か

入管対応やビザ更新の業務自体は大きな業務負荷ではないため、それ単体のために校長を置くことは効率が悪い。日本語学校の事務職は一般企業の人事・総務・経理に対応する業務があれば十分。

### 校長が必要な場面

**「学生への対応判断を最終的に行うポジション」** としての校長は必要。

- 日本語学校は「論理では割り切れない対象」（留学生）を扱うため、絶対的な正解がない場面が多い
- 判断を教師に委ねると、他の教師が異論を唱え内部摩擦が発生する
- 例: 学費未納の学生を退学にするかどうか → ある教師は見送り、別の教師は退学を主張
- **全体を俯瞰して意思決定できる「必要悪」としての校長** が求められる

## 生活指導員の役割

### 代表的な業務

- 留学生へのアルバイト紹介
- 病院の手配・付き添い対応
- 借り上げ寮の見回り・生活指導
- 居住トラブルの対応・清掃の促進
- 郵便物の管理、役所手続きのサポート
- 市役所対応（住民登録・転出届等）
- 学生証発行・再発行対応（地味だが頻繁に発生する）
- その他、学生の日本生活における雑務全般

要するに、**「日本語教師が担当する授業以外のあらゆる仕事」** が生活指導員の役割。

### 業務の特性

- 高度に専門的な知識やスキルは不要（ハローワークの求人応募者でも遂行可能）
- ただし「相手が外国人である」ことが重要。「日本人だったら分かるはず」という前提で接する人は不適任
- 日本語が通じない学生に根気強く寄り添う「共感性」と「柔軟性」と「粘り強さ」が求められる

## 人件費の構造

### 1クラスあたりのコスト試算

| 項目 | 金額 |
|------|------|
| 常勤講師（月給24万+社保）| 約28万円/月 |
| 非常勤講師（週1日、月4回）| 約3.2万円/月 |
| **合計** | **約31.2万円/月** |

### クラス人数と粗利の関係

| クラス人数 | 月額売上（@5.5万） | 人件費 | 粗利 |
|-----------|-------------------|--------|------|
| 10名 | 55万円 | 31.2万円 | 23.8万円 |
| 20名 | 110万円 | 31.2万円 | **78.8万円** |

::: warning 重要
1クラスあたりの人数は利益率に直結する最重要指標。学生のレベルに合わせた細分化を優先しすぎると少人数クラスが乱立し、収益性が大幅に低下する。講師にも事業運営の視点を理解させることが重要。
:::

### 常勤 vs 非常勤の利益効率

| パターン | 年間売上（20名/クラス） | 人件費 | 粗利 |
|---------|----------------------|--------|------|
| 常勤16コマ+非常勤4コマ | 1,200万円 | 440万円 | 760万円 |
| 全て非常勤（800コマ×2,000円） | 1,200万円 | 160万円 | 1,040万円 |

理論的には全非常勤が高効率だが、実務面では困難が伴い現実的ではない。
