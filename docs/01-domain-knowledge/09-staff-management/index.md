# 教職員管理

教員資格・配置の管理を行うカテゴリです。

## 概要

日本語学校の教員には法定の資格要件があります。

### 人的要件

| 役職 | 要件 | 配置数 |
|------|------|--------|
| 校長 | 教育業務5年以上従事 | 1名必須 |
| 教務主任 | 常勤日本語教員3年以上 | 1名必須 |
| 生活指導担当者 | 生活指導・進路指導の知識経験 | 1名以上 |
| 常勤講師 | 教員資格要件あり | 最低3名以上、教員全体の過半数 |
| 教員全体 | 学生20名につき1名以上 | — |

### 教員の資格要件（いずれか）

- 日本語教育を主専攻/副専攻として大学・大学院を卒業
- 日本語教師養成講座（420時間以上）修了
- 日本語教育能力検定試験合格
- ※ 新制度では「登録日本語教師」（国家資格）への移行中
