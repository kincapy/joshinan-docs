# 業務フロー — 月次学費管理サイクル

## フロー図

```mermaid
graph LR
    A[請求作成] --> B[請求書発行]
    B --> C[入金受付]
    C --> D[入金消込]
    D --> E[残高更新]
    E --> F[売上計上]
    F --> G[経理表出力]
```

## 各ステップの概要

| ステップ | 内容 |
|----------|------|
| 請求作成 | 学生ごとに月次の請求データを生成（seikyu-create） |
| 請求書発行 | PDF請求書を出力し、学生またはエージェントへ送付 |
| 入金受付 | 現金または口座振込で入金を受け付け |
| 入金消込 | 入金データを請求に紐づけて消込処理（nyukin-update） |
| 残高更新 | 入金後の残高を再計算 |
| 売上計上 | ステータスが「完了」の請求を売上として計上（uriage-create） |
| 経理表出力 | 月次の経理表を出力 |
