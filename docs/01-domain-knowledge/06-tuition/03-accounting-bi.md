# 会計構造・BI

学費管理の上位レイヤーとして、日本語学校全体の会計構造とBI（ビジネスインテリジェンス）の設計方針をまとめる。

## 会計構造

### 売上

| 売上科目 | 内容 |
|---------|------|
| 学費 | 学生からの授業料収入 |
| 寮費 | 学生からの寮費収入 |

### 売上原価

| 原価科目 | 対応する売上 | 備考 |
|---------|------------|------|
| エージェント手数料 | 学費 | 学生の入学時に一括で発生（月按分ではない） |
| 寮の家賃 | 寮費 | 会社として不動産オーナーに支払う家賃 |
| 水光熱費 | 寮費 | 物件ごとに月次で発生 |

### 粗利の特徴

::: warning エージェント費用の一括計上による粗利のブレ
エージェントへの手数料は学生が入学した月に一括で発生する（月按分ではない）。そのため入学月は粗利が大幅にマイナスになることがあり、これが日本語学校の収益構造上の特徴。
:::

## BI（ビジネスインテリジェンス）

### 1. 学生向けBI：残高推移

学生を選択したときに、月次の請求テーブルと入金テーブルから**学生ごとの月次残高推移**を表示する。

- 請求テーブルと入金テーブルは月次で存在するため、学生別の残高推移が算出可能
- 過払い・未収金の状態を時系列で把握できる

### 2. PL系：トップライン

| 指標 | 内容 |
|------|------|
| 売上推移 | 学費・寮費の月次推移 |
| 粗利の月次推移 | 売上 − 原価（エージェント費用 + 寮関連費用） |

::: info 粗利推移の注意点
エージェント費用は入学月に一括で計上されるため、入学月（4月・10月）に粗利が大きく下がる。月按分ではないことを前提にグラフを読む必要がある。
:::

### 3. BS系：現金回収率

年度末の想定売掛金額に対して、現時点での入金額の進捗率を表示する。

```
現金回収率 = 当年度の入金額 / 当年度の請求額 × 100
```

::: warning 年度単位で見るべき理由
翌年度の学費・寮費を先に支払っている学生がいる。例えば学生Aの入金額が120万円、年度内請求金額が110万円の場合、年度内の現金回収率は年度内単位で算出すべき。

ただし、学生個人別で見た場合は「卒業までの入金率（＝生涯入金率）」も有用な指標。
:::

### 4. その他のBI

| BI項目 | 内容 |
|--------|------|
| 国別在校生グラフ | 国籍別の学生構成を可視化 |
| 年度別・月別の学生数推移 | 積み上げ棒グラフで入学時期ごとの推移を表示 |
| 粗利の月次推移（概算） | 売上 − 原価のざっくりとした月次推移 |
