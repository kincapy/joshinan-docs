# 各提出書類の詳細注意事項

在留資格認定証明書交付申請における個別の提出書類ごとの注意事項。

## 各種確認書

- 日本語教育機関で作成することも可能
- 最下欄の申請人氏名は、代理人（学校職員）ではなく **申請人ご本人の氏名** を記名する（署名である必要はない）

## 最終学校の卒業証明書

::: info 対象
適正校である旨の通知を受けていない日本語教育機関のみ
:::

- 冊子型の卒業証書は表紙等の写しも提出
- 写し作成者の認証が必要（作成日・作成者氏名・申請人との関係）
- **卒業証書の原本確認は一括受付日に実施** — 原本を持参すること（非適正校のみ）

### 中国の教育機関の場合

| 学歴 | 推奨される提出書類 |
|------|-----------------|
| 大学（大専含む）卒業者 | CHSI又はCDGDCが発行する学歴等の認証報告原本 |
| 高中卒業（高考受験者） | CDGDCが発行する学歴等の認証報告原本 |

- CHSI: 全国高等学校学生信息諮詢職業指導中心
- CDGDC: 教育部学位及び研究生教育発展中心

::: tip 認証報告原本の代替
認証機関の認証結果画面ページを印刷したものも認証報告原本として取り扱われる。認証報告原本を提出せず受入校で真偽確認を行った場合は、その経緯と根拠資料を提出する。
:::

## 日本語能力立証資料【原本を提出】

::: info 対象
適正校II（別表掲載国以外）及び非適正校
- 適正校IIでも、外国の大学・短大・大学院を卒業しその卒業証明書を提出する場合は不要
:::

| 3 | J.TEST実用日本語検定 | 日本語検定協会・J.TEST事務局 | F級以上又はFGレベル250点以上 |
| 4 | 日本語NAT-TEST | 専門教育出版 | 5級以上 |
| 5 | STBJ標準ビジネス日本語テスト | 応用日本語教育協会 | 350点以上 |
| 6 | TOPJ実用日本語運用能力試験 | TOPJ実用日本語運用能力試験実施委員会 | 初級A以上 |
| 7 | J-cert生活・職能日本語検定 | 国際人財開発機構 | 初級以上 |
| 8 | JLCT外国人日本語能力検定 | 外国人日本語能力検定機構 | JCT5以上 |
| 9 | 実践日本語コミュニケーション検定・ブリッジ（PJC Bridge） | サーティファイ | C-以上 |

::: warning 試験結果の証明について
- 試験No.3, 4, 5, 6, 7, 9 については「出入国在留管理局提出用」の証明書（原本）を提出すること

- 一括受付日までに結果が出ない場合: 受験票写しを提出し、追加書類提出期限までに成績証明文書を提出
:::

### 日本語学習証明文書による立証

試験結果がない場合、現地日本語学校の学習証明文書で立証できる。以下の項目が記載されたものが望ましい。

| # | 記載項目 | 説明 |
|---|---------|------|
| 1 | 学習期間 | 受講した期間 |
| 2 | 出席率 | — |
| 3 | 履修内容 | — |
| 4 | コース名称 | — |
| 5 | 当該コースの終了目標レベル | — |
| 6 | 使用教材名 | — |

::: danger 重要
カリキュラムの総学習時間だけでなく、**実際に授業に出席した実学習時間数** が記載されていること。
:::

## 経費支弁書【原本を提出】

::: info 対象
適正校II（別表掲載国以外）及び非適正校
:::

申請人が日本入国後に発生する全ての経費（学費・生活費）について、経費支弁者が支弁金額と送金方法を具体的に示すもの。**経費支弁者本人が直筆又は署名** したものを提出する。

### 両親以外の者が経費支弁する場合

経費支弁書又は別紙で以下を説明する：
- 申請人の両親に代わり経費支弁を受諾した具体的経緯と事情
- 申請人の両親が経費支弁を履行できない具体的経緯と事情

### 奨学金による経費支弁の場合

**貸与型奨学金の場合:**
- 出入国在留管理庁HPの案内を確認の上、必要書類を提出

**給付型奨学金の場合:**
- 資料4別添4「奨学金の給付に関する証明書一覧」を参照

**共通:**
- 過去2年分の給付対象者名簿を提出（記載項目: 在籍校名称、支給開始時期、身分事項、在留カード番号）

## 経費支弁者と申請人との関係を立証する文書【原則原本】

::: info 対象
適正校II（別表掲載国以外）及び非適正校
:::

公的機関が認証した文書（出生証明書、親族関係公証書等）を提出する。

### 親族関係の場合

直接の関係だけでなく、中間の親族関係も全て立証する必要がある。

**例: 経費支弁者が申請人実母の姉の場合**
1. 申請人と実母の母子関係を立証する書類
2. 実母と経費支弁者（姉）の姉妹関係を立証する書類

::: danger 宣誓供述書のみでは不十分
親族関係立証に係る書類として、宣誓供述書のみの提出では不十分。公的機関の認証文書が必要。
:::

### 家族戸籍簿を提出する場合

世帯の住所と世帯構成員全員の身分事項・職業等が記載されたページの写しを提出する。

## 預貯金残高証明書【原本を提出】

::: info 対象
適正校II（別表掲載国以外）及び非適正校
:::

- **必ず原本を提出**（写しは不可）
- 発行日（立証日）から一括受付日までの期間が短いものが望ましい

## 過去1年間の資金形成過程立証資料【原則原本】

::: info 対象
適正校II（別表掲載国以外）及び非適正校
:::

預貯金残高証明書の金額を形成した経緯を立証する資料。**立証期間は残高証明書発行日を含めた過去1年間。**

### 預貯金通帳写しによる立証

以下の全項目が明確に判読できること：
- 全ページ（表紙・取引内容がないページも含む）がA4に収まっている
- 通帳の地模様が確認できる
- 取引内容（日付・金額・残高）がはっきり判読できる
- 預貯金者名・金融機関名・口座番号等の重要事項と銀行印が判読できる

::: warning 判読できない部分がある場合
理由書（任意様式）で該当部分について必ず説明すること。
:::

### 出入金明細書による立証

- 必ず **原本を提出**
- A4以外のサイズの場合は、原本と併せてA4に合わせた写しも提出
- 特に説明が必要な出入金は理由書で説明（原本への書き込みは不可）
- 理由書は経費支弁者等の事情を最も把握している人物が作成し、日付・署名を入れる

### 定期預貯金口座による立証

- 定期預貯金の証書に加え、当該口座に預貯金した金額の出所・資金を得るに至った資料が必要
- 例: 別の口座から資金移動した場合は、元の口座の通帳写し等

### その他の資料による立証

やむを得ない場合のみ。経費支弁者の収入状況について公的機関認証文書を必ず提出した上で、具体的な資金形成過程がわかる資料（例: 農産物の市場価格がわかる書類、売上伝票、領収書等）を併せて提出する。

## 経費支弁者の職業を立証する文書【原則原本】

::: info 対象
適正校である旨の通知を受けていない日本語教育機関のみ（非適正校）
:::

| 経費支弁者の職業 | 必要書類 |
|----------------|---------|
| 会社員 | 勤務先又は公的機関発行の在職証明書原本 |
| 事業者（自営業等） | 営業許可書写し・事業登録証明書写し等 |
| 企業の役員 | 公的機関発行の法人登記簿謄本等写し |
| 上記以外（農業従事者等） | 職業を立証できる公的機関発行の証明文書 |

## 過去1年間の経費支弁者の収入を立証する文書【原本】

::: info 対象
適正校である旨の通知を受けていない日本語教育機関のみ（非適正校）
:::

過去1年分の課税証明書（収入・所得）及び納税証明書（免税証明書）。

- 個人所得税を管轄する公的機関から発行された証明文書を提出
- 収入額は **総収入額** と **租税公課等控除後の実収入額** の双方が記載されたもの
- 納付済レシート等の写しではなく、公的機関が発行した証明文書を提出

### 農業従事者の場合の参考資料

経費支弁者が農業従事者（兼業含む。林業・水産業・畜産業含む）の場合、以下の資料が参考となる：

1. **農業規模**: 土地建物・家畜等の所有状況、登記簿等の写しと土地建物税の納税証明書
2. **従事人数**: 家族と雇用者を分けた人数
3. **生産設備**: 農業機械・トラック・厩舎等の設備設置状況
4. **販売契約者・販売経路**: 具体的な販売の仕組み
5. **過去1年間の収支状況**: 収入（生産物・生産量・販売単価等）と支出（土地税・設備維持費・肥料代・人件費等）
6. **過去1年間の月別売上状況**: 各月の売上高・主な生産物・生産量・販売数量・販売単価等

## 経費支弁者が在日親族等の場合

::: info 対象
適正校II（別表掲載国以外）及び非適正校
:::

| # | 必要書類 |
|---|---------|
| ア | 経費支弁書 |
| イ | 住民票（個人番号の記載なし・世帯全員分） |
| ウ | 直近1年分の住民税所得課税証明書＋納税証明書 |
| エ | 職業を証明する資料（在職証明書・営業許可書等） |
| オ | 残高証明書＋1年分の出入金明細書（通帳写し） |

::: tip 課税証明書の注意
控除や扶養について含まれるものを提出すること。市区町村によって証明書の名称が異なる場合がある。非課税の場合は納税証明書の提出は不要。
:::

## 18歳未満の申請人が単身で生活する場合

- 「単身」＝一人暮らしのこと。寄宿舎に入寮し寮母等がいる場合は含まない
- 日常的なサポートを誰がどのように行うか、日本語教育機関が確認した内容を説明する文書を提出

## 各種理由書【原本を提出】

### 理由書が必要となるケース

| ケース | 記載すべき内容 |
|--------|-------------|
| 在留資格認定証明書交付申請の不交付歴がある場合 | 事実の申告＋不交付理由が払拭されている立証 |
| 在留資格変更許可申請の不許可歴がある場合 | 事実の申告＋不許可となった理由の明確な説明 |
| 申請の取下歴がある場合 | 事実の申告＋取下理由＋理由が解消された状況の説明 |
| 在留資格認定証明書の返納歴がある場合 | 事実の申告＋返納理由＋理由が解消された状況の説明 |
| 交付を受けたが入国に至っていない場合 | 事実の申告＋入国に至らなかった理由＋解消された状況の説明 |

### 理由書の作成方法

- **申請人本人が作成** し、作成日を記載の上、署名（押印可）
- 不交付・不許可理由の払拭については、内容を裏付ける **具体的な立証資料** を必ず添付
- 理由書の作成者は、その内容に最も深く関与している者（申請人本人、書類発行元機関等）

## 申請人の職業を立証する文書【原本】

以下に該当する場合は、申請人の在職証明書を提出する：

- 過去に「留学」「就学」の不交付歴・不許可歴があり、申請時に職業を有する場合
- 過去に日本で「技能実習」（「研修」含む）を修了し、帰国後に在職歴がある場合
