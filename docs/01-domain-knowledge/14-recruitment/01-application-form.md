# 申請書作成の注意事項

在留資格認定証明書交付申請書（別記第6号の3様式）を作成する際の注意事項。入管から毎年配布される資料に基づく。

::: warning 最新様式を使用すること
申請書の様式は出入国在留管理庁ホームページで最新のものを確認すること。古い様式で提出すると差し戻しになる。
:::

## 申請番号の仕組み

### 学校番号と申請番号

- **学校番号**: 各日本語教育機関に配付された3桁の番号（原則変更なし）
- **申請番号**: 学校番号＋3桁の連番 ＝ 計6桁

| 項目 | 説明 | 例（学校番号100の場合） |
|------|------|----------------------|
| 学校番号 | 入管から配付された3桁 | 100 |
| 申請番号 | 学校番号＋001から連番 | 100001, 100002, ... |

### 申請番号の運用ルール

- 毎年1月にリセットされ、001から再スタート
- その年の最初の申請（7月期生以降）から翌年4月期生の申請まで連番で使用
- **取下げた申請番号は再利用できない**（使用済みとして扱う）

::: info 具体例
学校番号「100」の場合、毎年7月期生（4月期生の再申請がない場合）の最初の申請番号は「100001」から。
:::

## 申請書1枚目（申請人等作成用1）の記載

### 欄外への記載

申請書1枚目の右上部に以下を記載する：
- **申請年月日**
- **申請番号**（上記の6桁番号）

### 不交付歴がある場合

過去に在留資格認定証明書の不交付歴または在留資格変更不許可歴がある者は、申請年月日・申請番号の近くに **「不交付歴有」** と必ず記載する。

::: danger 重要
「留学」「就学」以外の在留資格に係る不交付歴・不許可歴も含む。漏れなく申告すること。
:::

### 顔写真の要件

| 項目 | 要件 |
|------|------|
| 撮影時期 | 提出日前6か月以内 |
| サイズ | 縦4cm × 横3cm |
| 条件 | 鮮明・無帽・無背景・正面向き |
| 裏面 | 申請番号を記載（剥離に備えて） |

::: warning 写真のNG例
- カラーコピーで作成した写真は使用不可
- 画像加工（目を大きく見せる、美白処理、ほくろ・しわの修正等）は不可
- 左右反転した写真は不可
- 宗教上・医療上の理由で着用物がある場合は、陳述書（任意様式）で事情を説明
:::

## 注意が必要な記載項目

申請書の記載漏れ・誤記載は毎年多数発生している。以下の項目は特に注意が必要。

### 氏名（項目3）

| ケース | 記載方法 |
|--------|---------|
| 旅券を持っている場合 | 旅券の表記どおりに **英字の大文字** で記載。漢字氏名は不要 |
| 旅券がない場合 | 本国の公的機関発行の資料に基づき、LAST → FIRST → MIDDLE の順番で記載 |
| 漢字圏で英字氏名が不明な場合 | 漢字氏名を記載し、申請後直ちに英字氏名を特定して追加資料を提出 |

::: danger 氏名の誤記載に注意
氏名の誤記載は毎年頻発している。誤った氏名で在留資格認定証明書が交付されると、査証発給手続に重大な支障が出る。申請書作成後に必ず再確認すること。
:::

### 出生地（項目5）

- 省略せず市区町村名まで記載する
- 中国・ベトナム国籍の場合: 「○○省○○市（県）」まで記載
- 出身省の誤記入が毎年多数発生しているため要確認

### 本国における居住地（項目8）

- 申請時点の居住地を地番まで省略なく記載
- 中国・ベトナム国籍の場合: 「○○省○○市（県）～○○号」まで記載

### 滞在予定期間（項目14）

- 受入校に設置されている告示コースの終期にあてはまること
- **途中退学となるような滞在予定期間は原則として認められない**

### 過去の出入国歴（項目17）

- 在留資格にかかわらず、日本に出入国歴がある場合は「有」を選択し、直近の出入国年月日を正確に記載
- 別の国籍や氏名変更前での出入国歴がある場合は、異なる国籍や氏名を **欄外に記載**
- 現在在留中の場合: 入国日から出国予定日を記載
- 再入国許可（みなし再入国許可含む）の出入国歴は計上不要

### 過去の在留資格認定証明書交付申請歴（項目18）

- 申請後に取り下げた場合も **「有」** を選択する

### 修学年数（項目23）と最終学歴（項目24）

**最終学歴の記載パターン:**

| 状況 | 在籍状況 | 学校名 | 卒業年月日 |
|------|---------|--------|----------|
| すでに卒業して就労中等 | 卒業 | 卒業した学校名 | 卒業年月日 |
| 在学中で入学までに卒業見込み | 在学中 | 在学中の学校名 | 卒業予定年月日 |
| 在学中だが入学までに卒業見込みなし | 卒業 | 直近の卒業校名 | その卒業年月日 |
| 高校卒業レベル検定試験合格者 | 卒業（中学校） | 中学校名 | 中学校卒業年月日 |

::: tip 欄外記載が必要なケース
- 在学中又は中退した学校名がある場合は **欄外に記載**
- 高卒検定試験合格者は欄外に「○年○月高卒検定試験合格」と記載
:::

### 日本語能力（項目26）

- **試験による証明**: 申請日までに結果が出た試験の結果を記載。結果が未判明の場合は受験日を試験名の横に記載
- **日本語教育を受けた教育機関及び期間**: 修了予定日ではなく申請日までの学習歴を記載し、欄外に修了予定日を記載。**実学習時間数** も記載すること

### 滞在費の支弁方法等（項目28）

#### 支弁方法及び月平均支弁額
- 入学時に一括納入する場合: 欄外に「入学金、授業料等○○万円については入学時に納入予定」と記載し、それ以外の滞在費の支弁方法と月平均支弁額を記載
- アルバイトを予定している場合: 主な支弁方法を記載した上で「その他」欄に予定月額報酬額と開始予定時期を記載

#### 経費支弁者
- 複数の経費支弁者がいる場合: ①○○、②○○と番号を付けて各々の情報を記載
- 職業欄: 「会社員」「自営業」ではなく「銀行勤務」「食品会社勤務」「ホテル経営」等、具体的な業種が分かるように記載
- 副業がある場合はその内容も記載
- 年収: **換金レートを記載の上、日本円で記載**。税引き後の年収額が確認できれば欄外に記載
- 同一経費支弁者が複数の申請者を支弁する場合: 「①氏名」の横に同一経費支弁者の申請番号を記載

## 所属機関等作成用の記載

### 法人名（項目2-3）
- 日本語教育機関を運営する法人（学校法人、株式会社等）の名称を記載

### 在籍区分（項目5）
- 学校法人として運営: 「日本語教育機関（各種学校）」を選択
- それ以外: 「日本語教育機関（その他）」を選択

### 仲介業者又は仲介者（項目9）

| 利用形態 | 記載内容 |
|---------|---------|
| 仲介業者 | 業者の名称と住所 |
| 仲介者（個人） | 仲介者の氏名と住所 |

::: warning ベトナムの仲介業者
ベトナムの仲介業者は本国政府から登録を受けた業者のみが仲介業を営むことができ、登録番号が付与される。
- 業者名に加えて **登録番号** を記載すること
- 入管から情報提供された番号と業者から回答のあった番号が異なる場合は、両方の番号を記載し、理由書を提出
- 登録番号が特定できない場合も空欄にせず、その旨を記載し理由書で説明
:::

## その他の留意事項

### 該当なしの場合
空欄にせず「**なし**」と記載する。

### 記載内容の訂正
- 訂正箇所に二本線を引いた上で、申請書作成者が訂正の署名をする
- **修正液・修正テープによる訂正は不可**

### 教育機関名・代表者氏名の記名
- 学校名と学校長等の **職名** 及び代表者氏名を記入
- 代表者氏名のみで職名が欠けるケースが多いので注意
