# 提出書類一覧と注意事項

在留資格認定証明書交付申請において、申請書以外に必要となる提出書類に関する全般的なルール。

## 提出書類全般のルール

### 書類の返却

提出された書類は **一切返却されない**。原本が提出可能な書類か（複数回発行できるかどうか）を必ず確認した上で提出すること。

### 書類の編綴方法

| 書類の厚さ | 綴じ方 |
|-----------|--------|
| 厚みがある | ガチャック又はダブルクリップ |
| 厚みがない | 書類の左上をステープラー止め |

::: danger ゼムクリップは使用不可
ゼムクリップで綴じた書類は受理されない。
:::

### 提出書類の有効期間

- 原則: **発行日から3か月以内** の書類を一括受付日に提出
- やむを得ない事情がある場合: 発行日から **6か月まで** の書類とし、事情を説明した理由書（任意様式）を提出
- 説明内容によっては立証資料として取り扱われない場合がある

### 日本語訳の添付

外国語（**英語を含む**）で作成された提出書類には、原則として書類ごとに日本語訳を添付する。

| 項目 | ルール |
|------|--------|
| 固有名詞 | 英語（アルファベット表記）のままで可 |
| 本文 | 日本語訳を必ず添付 |
| 出入金明細書 | 取引金額以外の記載箇所（取引種別等）も日本語訳が必要 |
| 英語以外の外国語 | 原文と日本語訳の両方を提出（英訳のみは不可） |

::: warning 誤訳に注意
日本語訳も参考にして審査が行われるため、誤訳があると審査結果に影響する。
:::

### 各種証明書の発行先機関

以下の情報が記載された証明書が望ましい：
- 発行者の肩書及び氏名
- 発行機関の所在地
- 連絡先（電話、FAX番号、E-mailアドレス）
- 可能な限り発行者の署名（押印可）

### A4サイズ以外の証明書

| 原本の状態 | 対応方法 |
|-----------|---------|
| A4より小さい | 原本はそのまま提出＋A4に拡大した片面コピーを添付 |
| A4より大きい | A4上辺に合わせて谷折り＋A4に縮小した片面コピーを添付 |
| A4の両面印刷 | 原本を提出＋A4片面でコピーしたものを添付 |

### 写しを提出できる場合

**写しの提出が認められるケース:** 戸籍簿、卒業証書等、複数回発行できない書類のみ。

預貯金残高証明書や日本語学習証明文書など、複数回発行可能な書類は **原本を提出** すること。

**写し作成時の留意事項:**

| 要件 | 内容 |
|------|------|
| コピー方法 | 原本を直接複写した一次コピーのみ |
| 印刷方法 | 片面印刷 |
| サイズ | A4サイズ（原本がA4以外でもA4で全体が収まるように） |
| 認証 | 余白部分に「①写し作成日、②作成者氏名、③申請人との関係」を記載 |

記載例：`写し作成日 令和○年○月○日 作成者 ○○○○（受入校職員）`

### 参考資料として写真を提出する場合

- 写真の原本と、A4用紙に印刷した同写真の写しの両方を提出
- 提出された写真原本は返却不可

## 提出書類の添付順番

申請書に提出書類を添付する場合は、以下の順番で編綴する。日本語訳文は外国語原文に添付する。

| 順番 | 書類 |
|------|------|
| ① | 申請書 |
| ② | 提出書類一覧表及び各種確認書 |
| ③ | 旅券の写し |
| ④ | 最終学校の卒業証明書 |
| ⑤ | 日本語能力立証資料 |
| ⑥ | 経費支弁書 |
| ⑦ | 経費支弁者と申請人との関係を立証する文書 |
| ⑧ | 預貯金残高証明書 |
| ⑨ | 過去1年間の資金形成過程を明らかにする資料 |
| ⑩ | 経費支弁者の職業を立証する資料 |
| ⑪ | 過去1年間の経費支弁者の収入を立証する資料 |
| ⑫ | 奨学金の給付に関する文書 |
| ⑬ | 18歳未満の申請人が単身で生活する場合の体制等を証する資料 |
| ⑭ | 各種理由書及び裏付け資料 |
| ⑮ | その他、任意で提出する資料 |
| ⑯ | 結核非発病証明書 |

## 各種書類の保管

学生募集時に使用した各種書類（学生募集要項、パンフレット等）は、審査に必要な場合に入管が提出を求めることがある。**最低3年間** は日本語教育機関で適切に保管すること。
