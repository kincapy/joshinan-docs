# 募集業務

日本語学校で事業運営上最も重要な「募集業務」についてまとめたカテゴリです。日本語学校は授業料モデルのため、在籍数が重要な経営指標であり、事業を安定して運営するには募集業務の土台を作る必要があります。

## 募集業務概要

### スケジュール

留学生の入学時期は基本的に4月と10月。

| 入学時期 | 書類申請 | ビザ結果 | 入国 |
|---------|---------|---------|------|
| 4月生 | 前年11月頃 | 2月 | 3月後半〜4月前半 |
| 10月生 | 5月頃 | 8月 | 9月後半〜10月前半 |

### スケジュールの詳細（4月生の場合）

| 時期 | 作業内容 |
|------|---------|
| 8〜9月 | 面接実施 |
| 10月頃 | 現地から書類を送ってもらう |
| 10月末 | 書類の順番を学校で揃える |
| 11月 | 入管に書類申請 |
| 2月 | ビザの結果判明 |
| 2月〜3月 | ビザ交付後の事前面接 |
| 3月後半〜4月前半 | 入国 |

::: warning 早すぎる募集は意味がない
外国人は他の選択肢（別の海外・他の日本語学校）を持っているため、1年前から募集しても候補者が1年間ずっと勉強することはほぼあり得ない。募集を早く動けばいいというわけではない。
:::

## 適正校制度と書類の簡素化

入管は日本語教育機関を「適正校」と「非適正校」に分類している。適正校の認定を受けると提出書類が大幅に簡素化される。

### 適正校の分類

| 分類 | 対象 | 書類の簡素化レベル |
|------|------|-----------------|
| 適正校クラスI | 在籍管理が特に良好な機関 | 最も簡素（別表掲載国と同等） |
| 適正校クラスII | 在籍管理が良好な機関 | 別表掲載国は簡素、それ以外は一部必要 |
| 非適正校 | 上記以外 | 全書類が必要 |

::: warning 適正校の認定基準
適正校の認定は入管が毎年行う。不法残留者率などの在籍管理状況が基準となる。一度認定されても翌年に外れることがある。
:::

### 適正校クラスII の提出書類一覧

実務上最も多いケース。別表掲載国かどうかで提出書類が異なる。

| # | 書類 | 別表掲載国 | 別表掲載国以外 | 備考 |
|---|------|:--------:|:----------:|------|
| 1 | 在留資格認定証明書交付申請書 | ○ | ○ | — |
| 2 | 提出書類一覧表・各種確認書 | ○ | ○ | — |
| 3 | 旅券の身分事項ページ（写し）| ○ | ○ | 追記欄に記載がある場合はそのページも |
| 4 | 日本語能力に係る資料 | — | ○ | 外国の大学等を卒業しその証明書を提出する場合は不要 |
| 5 | 経費支弁書 | — | ○ | 経費支弁者本人が直筆又は署名したもの |
| 6 | 経費支弁者と申請人の関係を立証する資料 | — | ○ | 出生証明書・親族関係公証書等 |
| 7 | 預金残高証明書（原本） | — | ○ | 経費支弁者の残高証明書 |
| 8 | 過去1年間の資金形成経緯を明らかにする資料 | — | ○ | 通帳の写し・出入金明細書等 |
| 9 | 奨学金の給付に関する証明書 | — | △ | 奨学金を受ける場合のみ |
| 10 | 18歳未満の申請人が単身で生活する場合の体制を証する資料 | △ | △ | 入寮する場合は不要 |
| 11 | 認定不交付・在留不許可の処分理由を払拭する資料 | △ | △ | 該当者のみ |

※ ○＝必須、△＝該当者のみ、—＝不要

::: info 非適正校の場合
上記に加えて以下の書類も必要になる：
- 最終学校の卒業証明書
- 経費支弁者の職業を立証する文書
- 過去1年間の経費支弁者の収入を立証する文書（課税証明書・納税証明書）
- 資金形成経緯は過去3年間分が必要
:::

::: tip 現地語の書類は日本語訳が必須
外国語（英語を含む）で作成された書類には、原則として書類ごとに日本語訳を添付する。誤訳があると審査結果に影響するため注意が必要。出入金明細書の取引種別など、金額以外の記載箇所も日本語訳が必要。
:::

## 交付率について

留学生のビザの交付率はかなり入管の匙加減に依存している。全く同じ書類を出した場合でも、ある人は通って別の人は落ちるということが普通に起こる。

### 構造的な課題

- 交付率の不安定さが日本語学校の事業を不安定にする
- 交付率が悪いとエージェントが学生に日本語を教えるインセンティブが低下
- 結果として日本語ができない留学生が発生する

::: tip 地域差
上記は主に東京入管の場合。他の地域の入管だともっと安定している場合もある。入管の匙加減に依存しない募集戦略を作ることが重要。
:::

## 募集戦略

### 別表掲載国

入管が指定する「別表掲載国」は、ビザ書類が簡素化されており、他の国で交付率が悪くても基本的にすべて交付される。

**アジアの別表掲載国**:
韓国、キプロス、サウジアラビア、シンガポール、タイ、台湾、中国、中国（香港）、中国（マカオ）、トルコ、ブルネイ、マレーシア、モルディブ

それ以外にアメリカ・ヨーロッパ諸国も含まれる。

::: tip 安定した募集のために
ベトナム、ネパール、スリランカなど一般的な国に加えて、別表掲載国の割合をいかに増やすかが募集安定化の鍵。
:::

### 募集の仕方

#### 面接のタイミング

- 申請前に面接は必要だが、面接時点では入学が先のため日本語力は高くない
- ビザの交付率が流動的なため、面接したけどビザが降りないことが日常茶飯事
- **申請前段階で時間をかけた面接をしても無駄骨になる可能性が高い**

#### 入国時の日本語レベルの限界

- 現地でN3を取るには最低1年近くの学習が必要
- しかし外国人からすると、そこまで勉強しなくても日本に行く手段がある（技能実習・特定技能）
- 入国時にN3は構造上ほぼ不可能。高いレベルでN4、平均的にはN5取得が及第点

#### ビザ交付後の事前面接（重要）

**募集活動の目的**:
1. 申請数を最大化し入学者を最大化すること
2. **入国前に聞いた情報と実際に来た後の情報のギャップをなくすこと**

ビザ交付後の事前面接で説明すべき内容:

| 項目 | 内容 |
|------|------|
| 学費の説明 | 金額・支払いスケジュール |
| 学校の注意事項 | 規則・ルール |
| ビザ更新の注意 | 出席率の重要性・週28時間制限 |
| 寮の注意事項 | 生活ルール |

::: warning 誓約書の重要性
- 説明は本人と経費支弁者（親）も含めて実施すべき
- 本人の署名付きの誓約書を締結することが極めて重要
- 留学生は書面にしないと「聞いていない」と言う。書面にしてもちゃんと読んでいない場合も多い
- トラブル時に「学校は事前に説明している」ことを証明するために必要
:::
