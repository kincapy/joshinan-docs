# 募集業務 データ定義

## 募集期 (RecruitmentCycle)

入学時期ごとの募集サイクルを管理する。4月生・10月生の年2回。

| 項目 | 内容 | 備考 |
|------|------|------|
| 入学時期 | 4月 / 10月 | 年2回の募集 |
| 年度 | 西暦（例: 2026） | |
| 書類申請締切日 | 4月生: 前年11月頃、10月生: 5月頃 | 入管への申請期限 |
| ビザ結果予定日 | 4月生: 2月、10月生: 8月 | |
| 入国開始日 | 4月生: 3月後半、10月生: 9月後半 | |
| 募集目標人数 | 各募集期の目標 | 経営計画に基づく |
| 申請数 | 実際に入管へ申請した人数 | |
| 交付数 | ビザが交付された人数 | 交付率 = 交付数 / 申請数 |

## 申請ケース (ApplicationCase)

個々の入学候補者に対する在留資格認定証明書（COE）の申請単位。

| 項目 | 内容 | 備考 |
|------|------|------|
| 募集期 | どの募集期に属するか | |
| 候補者氏名 | 学生候補者の氏名 | パスポート記載名 |
| 国籍 | 候補者の国籍 | |
| エージェント | どのエージェント経由か | nullable（直接募集の場合） |
| 申請番号 | 6桁の番号 | 入管が付与。申請後に判明 |
| 申請ステータス | 書類準備中 / 申請済 / 交付 / 不交付 / 取下 | 下記参照 |
| 別表掲載国フラグ | 別表掲載国に該当するか | 必要書類の判定に使用 |
| 交付日 | ビザ交付日 | 交付時のみ |
| 不交付理由 | 不交付の場合の理由 | 不交付時のみ |
| 学生ID | 交付後に学生として登録されたらリンク | 交付→入学の連携 |

### 申請ステータスの流れ

```
書類準備中 → 申請済 → 交付 → （学生として登録）
                    → 不交付
           → 取下
```

- **書類準備中**: エージェントから書類を収集している段階
- **申請済**: 入管に書類を提出済み
- **交付**: ビザが交付された（入国可能）
- **不交付**: ビザが交付されなかった
- **取下**: 何らかの理由で申請を取り下げた

## 申請書類 (ApplicationDocument)

申請ケースに紐づく個別の提出書類。

| 項目 | 内容 | 備考 |
|------|------|------|
| 申請ケース | どの申請ケースの書類か | |
| 書類種別 | 申請書 / 旅券写し / 日本語能力資料 / 経費支弁書 / ... | 下記参照 |
| 収集状態 | 未受領 / 受領済 / 確認済 / 不備あり | 下記参照 |
| ファイルパス | アップロードされたファイルの保存先 | |
| 日本語訳有無 | 日本語訳が添付されているか | 外国語書類には訳文が必要 |
| 備考 | 不備内容や特記事項 | |

### 書類種別の一覧

| 種別 | 対象 | 備考 |
|------|------|------|
| 在留資格認定証明書交付申請書 | 全申請者 | |
| 提出書類一覧表・各種確認書 | 全申請者 | |
| 旅券写し | 全申請者 | |
| 日本語能力資料 | 別表掲載国以外 | 適正校クラスIIの場合 |
| 経費支弁書 | 別表掲載国以外 | 適正校クラスIIの場合 |
| 経費支弁者との関係立証資料 | 別表掲載国以外 | 適正校クラスIIの場合 |
| 預金残高証明書 | 別表掲載国以外 | 適正校クラスIIの場合 |
| 資金形成経緯資料 | 別表掲載国以外 | 適正校クラスIIの場合 |
| 奨学金証明 | 該当者のみ | |
| 未成年者の経費支弁に関する補足 | 該当者のみ | |
| 理由書 | 該当者のみ | |
| その他 | 必要に応じて | |

### 収集状態の流れ

```
未受領 → 受領済 → 確認済
                → 不備あり → 受領済（再提出後）
```

- **未受領**: まだ書類を受け取っていない
- **受領済**: 書類を受け取ったが未確認
- **確認済**: 内容を確認して問題なし
- **不備あり**: 確認の結果、不備が見つかった（再提出が必要）

## 書類チェック結果 (DocumentCheckResult)

申請書類に対するチェック結果。人手またはAIによるチェックを記録する。

| 項目 | 内容 | 備考 |
|------|------|------|
| 申請書類 | どの書類に対するチェックか | |
| チェック種別 | 網羅性 / 訳文有無 / 並び順 | 下記参照 |
| 結果 | OK / NG | |
| 指摘事項 | NGの場合の具体的な指摘内容 | |

### チェック種別

| 種別 | チェック内容 |
|------|-------------|
| 網羅性 | 必要な書類が全て揃っているか |
| 訳文有無 | 外国語書類に日本語訳が添付されているか |
| 並び順 | 入管提出時の書類の並び順が正しいか |

## 識別子

| ID | 形式 | 用途 |
|----|------|------|
| 申請番号 | 6桁数字 | 入管が付与する申請の識別番号 |

## 必要書類の判定ロジック

適正校分類と別表掲載国フラグの組み合わせで、必要書類が決まる。

| 適正校分類 | 別表掲載国 | 必要書類 |
|-----------|-----------|---------|
| クラスI | - | 最も簡素（申請書・旅券写し等の基本書類のみ） |
| クラスII | 該当 | 基本書類のみ |
| クラスII | 非該当 | 基本書類 + 日本語能力資料 + 経費関連書類 |
| 非適正校 | - | 全書類必要 |

::: info 当校の現状
常南国際学院は適正校クラスIIのため、別表掲載国か否かで必要書類が分かれる。
:::
