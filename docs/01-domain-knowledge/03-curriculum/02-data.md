# カリキュラム・時間割 データ定義

## コース (Course)

日本語学校が提供する教育コースの基本情報。入管への届出やカリキュラム設計の基礎単位となる。

### 項目

| 項目 | 値の例 | 備考 |
|------|--------|------|
| コース名 | 進学2年コース | 届出上のコース名称 |
| 対象レベル（JLPT） | N3 | 入学時点の想定到達レベル |
| コース期間（月数） | 24 | コホートにより異なる（12〜24ヶ月） |
| 年間授業時間 | 800 | 法定基準: 760時間以上 |
| 有効期間 | 2024年4月〜2026年3月 | コースの運用期間 |

## 科目 (Subject)

授業で扱う個別の教科。時間割のセルに割り当てる単位。

### 項目

| 項目 | 値の例 | 備考 |
|------|--------|------|
| 科目名 | 文法 / N3文法 / みん日中級1 | RINGUALに登録されている名称 |
| カテゴリ | 文法 / 読解 / 聴解 / 会話 / 漢字 / 作文 / その他 | 科目の分類 |
| 対象レベル | N3 | 対象とするJLPTレベル（省略可） |

## 時間割 (TimetableSlot)

クラス × 曜日 × 時限で一意に決まる授業枠。科目と担当教員を割り当てる。

### 項目

| 項目 | 値の例 | 備考 |
|------|--------|------|
| クラスID | A1 / P2 | 午前クラス（A1〜A4）、午後クラス（P1〜P3） |
| 曜日 | 月 | 月〜土（日曜含む7曜日分の枠あり） |
| 時限 | 1限 | 時限マスタから参照 |
| 科目ID | 文法 | 科目マスタから参照 |
| 担当教員ID | 山田先生 | 教員マスタから参照（未定の場合あり） |
| 学期・年度 | 2024年度 前期 | 年度と学期の組み合わせで管理 |

### RINGUALでのセルID体系

RINGUALの内部では、時間割の各セルを `sel-tt-{曜日番号}_{時限番号}` で管理している。

| 曜日番号 | 曜日 |
|---------|------|
| 0 | 日 |
| 1 | 月 |
| 2 | 火 |
| 3 | 水 |
| 4 | 木 |
| 5 | 金 |
| 6 | 土 |

## 時限マスタ (Period)

1日の授業時間帯の区切り。午前・午後で時間帯が異なる。

### 項目

| 項目 | 値の例 | 備考 |
|------|--------|------|
| 時限番号 | 1 | 通し番号 |
| 開始時刻 | 09:00 | HH:mm 形式 |
| 終了時刻 | 09:45 | HH:mm 形式 |
| 時間帯区分 | 午前 / 午後 | 午前クラス・午後クラスに対応 |
