# カリキュラム・時間割

コース → 科目 → 授業の階層でカリキュラムを管理するカテゴリです。

## 法定要件

| 項目 | 基準 | 備考 |
|------|------|------|
| 年間授業時間 | 760時間以上 | 実際は800時間が多い |
| 週あたり授業時間 | 20時間以上 | — |
| 年間授業週数 | 38週以上 | — |
| 1クラス最大人数 | 20名 | 告示基準 |

## 時間割の構造

RINGUALでは、時間割は **クラス × 曜日 × 時限** の三次元で管理されます。

| 軸 | 内容 |
|------|------|
| クラス | A1〜A4（午前）、P1〜P3（午後）※年度ごとに設定 |
| 曜日 | 月〜土（日曜含む7曜日分の枠あり） |
| 時限 | 1限〜複数時限 |

### 科目の例

RINGUALに登録されている科目：

- 文法
- N3文法
- みん日中級1
- その他（年度・クラスレベルに応じて設定）

### 時間割セルのID体系

RINGUALの内部では、時間割の各セルを `sel-tt-{曜日番号}_{時限番号}` で管理しています。

| 曜日番号 | 曜日 |
|---------|------|
| 0 | 日 |
| 1 | 月 |
| 2 | 火 |
| 3 | 水 |
| 4 | 木 |
| 5 | 金 |
| 6 | 土 |
