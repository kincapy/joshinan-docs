# 学校基本情報

常南国際学院の基本情報とビジネスモデルをまとめたカテゴリです。

## ビジネスモデル

### 概要

日本語学校のビジネスモデルは「学校ビジネス」に該当し、主な収益源は留学生から徴収する授業料。学校がアパートを用意している場合は家賃収入も付随する。

**売上の重要指標は「在籍生徒数」のみに依存している。**

### 授業料の目安（地方校の場合）

| 項目 | 金額 |
|------|------|
| 入学金 | 約5万円 |
| 年間授業料 | 約70万円（税込） |
| 学生寮（相部屋） | 約25,000円/月 |

::: info 補足
都市部（特に東京都内）の学校は地方より授業料が高い傾向がある。教材費・設備維持費を別途徴収する学校も存在する。
:::

### 主な売上と販管費

**売上**: 授業料 + 家賃収入

**主な販管費**:

| 項目 | 詳細 |
|------|------|
| 人件費 | 校長・常勤講師・非常勤講師・事務員の給与 |
| 教材費 | 1名あたり年間約1.5万円 |
| 雑費 | 学校イベント（遠足・地域交流・バス代等） |
| **紹介手数料** | エージェントへの1人あたり紹介料。**販管費の中で最重要コスト要素** |

### 入学時期と収益性

**何月生として入れるかが最も大きな影響を持つ**:

| 入学時期 | 在籍期間 | 総売上 | 紹介料20万の場合の粗利 | 粗利率 |
|---------|----------|--------|---------------------|--------|
| 4月生 | 2年間 | 140万円 | 120万円 | 約85% |
| 10月生 | 1.5年間 | 105万円 | 85万円 | 約80% |

::: warning 会計上の注意（3月決算の場合）
10月生20名入学の場合、年度内の授業料収入は6ヶ月分=700万円。紹介料400万円+固定費で一時的に決算が悪化する。**4月生でいかに多く募集できるかが極めて重要。**
:::

### 紹介手数料の考え方

- 10月生の方が在籍期間が短いのに同じ手数料を払うのは不合理に見える
- しかしエージェント側は4月生でも10月生でも対応内容は同じ
- 海外では6-7月が学期末のため10月生の方が集客しやすい
- 手数料値下げ要求はエージェントにとってナンセンス
- **紹介手数料の多寡より「質の良い学生を安定的に送り出してくれるエージェント」との関係構築が本質**

### 主要指標

#### 募集人数

- ビザの交付率を前提にした設計が必要
- 例: 定員100名 × 交付率80% = 最低120名の募集が必要
- 入管制度上「申請可能数は欠員数の1.2倍まで」の制限あり

#### 1クラスあたりの人数

- 在籍人数が売上を規定するのに対し、**1クラスあたりの人数は利益率に直結する最重要指標**
- 詳細は [09 教職員管理](/01-domain-knowledge/09-staff-management/) の人件費構造を参照

### 理想的なビジネスモデル

授業料のみでの収益構造には限界がある（100名在籍で売上約6,000万円、営業利益約1,200万円程度）。

**留学生の経済的価値を複層的に活用する3つの事業モデル**:

#### (1) 人材派遣事業

- 留学生は在学中に週28時間まで就労可能
- 企業に派遣し、時給に300〜400円を上乗せして請求
- 1名あたり人月4万円程度の粗利
- **メリット**: 学生の収入把握→学費管理が容易 / 就業先管理→失踪リスク低減

#### (2) 人材紹介事業

- 卒業後の留学生に対し企業就職を斡旋
- 外国人採用に不慣れな企業への紹介料は20〜40万円程度
- 特定技能で紹介する場合、登録支援機関として月額1〜2万円の支援料（取得難易度は低い）

#### (3) 請負事業

- 物流系作業や野菜のカット業務など、業務単位で作業を受託
- 自社運営で留学生を投入するモデル
- 時給計算より業務単位での粗利が確保しやすい

::: tip 地方校にとっての重要性
アルバイト確保ができなければ学費未回収リスクが顕在化し、学生数の増加がむしろ経営悪化につながる可能性がある。アルバイトをコントロールできる環境（派遣・請負など）を整備できれば、定員増加に踏み切れる。
:::
