# 法令・基準の変遷

入管法改正や認定制度への移行など、日本語教育機関に関わる法令・基準の変遷を記録するカテゴリです。

## 制度変遷の年表

| 年 | 出来事 |
|----|--------|
| 1990年 | 日本語教育機関の告示制度が開始。法務省が「告示校」を管理 |
| 2016年7月 | 日本語教育機関の告示基準を策定（平成28年） |
| 2018年7月 | 告示基準を改定（平成30年） |
| 2019年6月 | 「日本語教育の推進に関する法律」成立。国の責務として日本語教育推進を位置づけ |
| 2019年8月 | 告示基準を大幅改定（令和元年）。出席率基準の厳格化、学習成果基準の新設 |
| 2023年5月 | 「日本語教育機関認定法」が国会で成立 |
| 2023年6月 | 認定法 公布（令和5年法律第41号） |
| **2024年4月** | **認定法 施行。認定日本語教育機関制度が開始。登録日本語教員制度が開始** |
| 2024年10月 | 第1回認定結果公表（72件中22件認定） |
| 2025年3月 | 第2回認定結果公表（48件中19件認定） |
| 2025年5月 | 告示基準の変更届出要件が改定（認定制度に合わせた教育課程の詳細記載を要求） |
| 2025年6月 | 告示基準改定（令和7年。第48号まで拡大） |
| 2025年10月 | 第3回認定結果公表（74件中23件認定） |
| **2029年3月末** | **告示校の移行期限。認定を受けないと留学生受入れ不可** |
| **2033年3月末** | **登録日本語教員の経過措置ルートCの期限** |

## 告示校 → 認定日本語教育機関への移行

### 制度比較

| 項目 | 告示校（旧制度） | 認定日本語教育機関（新制度） |
|------|-----------------|--------------------------|
| 所管 | 法務省（入管庁） | 文部科学省 |
| 根拠 | 出入国管理及び難民認定法の告示（法律に基づかない） | 日本語教育機関認定法（法律に基づく） |
| 教員資格 | 告示基準の要件（検定試験合格、420時間養成講座等） | 登録日本語教員（国家資格） |
| 認定基準 | 告示基準（48号） | 認定基準（文科省令） |
| 認定の更新 | なし（継続的に要件を満たせばよい） | 5年ごとの更新制 |
| 教育方法 | 特に指定なし（文型積み上げ方式が主流） | CAN-DO型式・CEFRベースの到達目標 |
| 課程の類型 | なし | 留学・就労・生活の3類型 |
| 自己点検評価 | 年1回（告示基準第18号） | 年1回以上の実施・結果の公表義務 |
| 情報公表 | 特になし | 教育目的・学習成果等の公表義務 |
| 報告先 | 入管庁のみ | 文科省＋入管庁（ビザ関連） |
| 移行期限 | — | 2029年3月末 |

### 移行の影響

- **教育課程の再設計**：CAN-DO記述文による到達目標の設定が必要。代表的教科書「みんなの日本語」は文型積み上げ方式のためCAN-DO型式に不向き
- **教員資格の取得**：既存教員は経過措置期間内に登録日本語教員の資格を取得する必要がある
- **自己点検評価の整備**：10項目（教育理念・運営・情報公開・入学選考・教育活動・教職員養成・学生支援・施設設備・地域貢献・財務）の評価を毎年実施・公表
- **二重報告体制**：移行完了まで入管庁への告示基準報告と文科省への認定基準報告の両方が必要

## 令和元年（2019年）告示基準改定の詳細

日本語学校の在籍管理厳格化のために行われた大幅な改定です。

### 主な変更点

| 項目 | 旧基準 | 新基準 | 影響 |
|------|--------|--------|------|
| 適正校判断基準 | 不法残留者のみ（3%以内） | 不法残留＋不許可＋取消＋資格外活動取消（5%以内） | 対象範囲が拡大 |
| 出席率（抹消基準） | 5割以下で抹消 | **7割以下**で抹消 | 大幅に厳格化 |
| オーバーステイ（抹消基準） | 半数以上で抹消 | **3割以上**で抹消 | 大幅に厳格化 |
| 学習成果基準 | なし | 進学等が修了者の**7割以上**必要 | 新設 |
| 出席率報告 | なし | 6か月ごとに年2回報告義務化 | 新設 |
| 定期点検報告 | なし | 毎年6月末に報告 | 新設 |

### 背景

留学生の在籍管理問題（所在不明者の増加、不法就労等）が社会問題化したことを受け、日本語教育機関への管理責任を強化するために改定が行われました。

## 認定法の構成

日本語教育機関認定法（令和5年法律第41号）は全4章26条で構成されています。

| 章 | 条文 | 内容 |
|----|------|------|
| 第1章 総則 | 第1条〜第2条 | 目的、定義 |
| 第2章 認定 | 第3条〜第11条 | 認定基準、有効期間（5年）、報告徴収、自己点検評価、取消し |
| 第3章 教員資格 | 第12条〜第22条 | 登録日本語教員の試験、登録、欠格事由 |
| 第4章 罰則 | 第23条〜第26条 | 最高で3年以下の懲役もしくは300万円以下の罰金 |
