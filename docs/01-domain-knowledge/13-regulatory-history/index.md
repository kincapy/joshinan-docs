# 法令・基準の変遷

入管法改正や認定制度への移行など、法令・基準の変遷を記録するカテゴリです。

## 主な変遷

### 告示校 → 認定日本語教育機関への移行

- **従来**：「告示」ベースの制度（法律に基づかない）
- **新制度**：法律に基づく「認定日本語教育機関」（2024年5月制定）
- **教育方法**：文型積み上げ方式 → CAN-DO型式への転換
- **課題**：代表的教科書「みんなの日本語」がCAN-DO型式に不向き

::: info 準備中
法令変遷の詳細な年表は今後追加予定です。
:::
