# 学生管理

学生の入学から卒業までのライフサイクルを管理する、最重要カテゴリです。

## 概要

- 現在の在籍学生数：238名
- 8ヶ国からの学生受入れ（中国が中心、モンゴル、ベトナム、ネパール、スリランカ等）
- 入学コホート：4月生 / 7月生 / 10月生 / 1月生
- 基本在籍期間：2年間または1.5年間
