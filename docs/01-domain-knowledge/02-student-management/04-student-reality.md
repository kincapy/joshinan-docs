# 留学生の実態

留学生がどういう存在で、なぜ日本に来て、どのような生活をしているかの実態をまとめたページです。日本語学校を理解する上で不可欠な知識です。

## なぜ留学するのか

### 留学生の実像

大前提として、「留学」と聞くと優秀な人が来ているように思えるが、それは誤解である。

- 日本人が海外に留学する場合 → 社会的に上の層（大学関係者、高学歴大学生、高所得者）が中心
- 外国人が日本に留学する場合 → 自国に留まっても給料が安い、身の回りの人が行っているから自分も、といった動機がほとんど

**日本はあくまで選択肢の一つ**:

- 社会的に上の層はアメリカやヨーロッパ圏を選ぶことが多い
- 日本に来る層は高卒が多い
- 表向きは「日本は経済的に成長していて〜」「日本の文化が好きで〜」と言うが、本質的には数ある選択肢の一つ

### 留学生の特徴

- 特に明確な目的もなく日本に来ている場合がほとんど（周りが行っているから自分も）
- 大学卒の割合は少なく、ほとんどが高卒
- 家庭もそこまで裕福ではないため、アルバイトは不可欠
- ただし、若くして言語も文化も全く異なる国に来ていること自体は立派であり、敬意を持って接すべき

::: tip まとめ
「なぜ留学するのか？」→ 自国に留まっても将来がなく、数ある選択肢の一つとして、知り合いや身の回りの人が多く行っているから日本を選んだ、というのが大半の理由。
:::

## 留学生が日本に来るまで

### 留学の要件

| 要件 | 内容 |
|------|------|
| 学歴 | 12年以上の学校教育修了者（高卒以上） |
| 日本語学習歴 | 150時間以上、またはJLPT N5以上 |
| 経歴 | 不自然な空白期間がないこと |

::: warning 実態
日本語学習時間にはばらつきがあり、ひらがな・カタカナも書けないまま留学してくる人もいる。学習証明書は現地の語学学校で作成してもらえるが、偽造も可能。
:::

### 来日までの典型的な流れ

1. 外国人が「自国に留まっても将来がない」と考える / 親から海外に出るよう言われる
2. 知り合い経由で聞いてみる、または現地の語学学校に登録する
3. 知り合いの知り合い（ブローカー/仲介人）に会い、日本留学の手続きをしてもらう
4. または語学学校が主に日本に人を紹介しているため、そこ経由で手続き

**日本語学習の強制力が弱い理由**:

- ブローカーは本人を送り出すことが仕事であり、日本語教育の義務はない
- 本人が留学ビザを取得して日本に来れば、ブローカーは本人からお金を取れる
- 結果として、全く日本語を話せないのに留学してくる学生が生まれる

::: info 補足
もちろん、しっかり日本語教育をする現地機関やブローカーも存在する。問題は「構造的に教育しなくても送り出せてしまう」こと。
:::

## 留学生の日常生活

- 半日授業（年間190〜200日）で夏休み・冬休みもあるため、比較的自由な生活
- 資格外活動許可で週28時間のアルバイトが可能
- 長期休みは通常の日本人労働者と同じ扱い（週28時間制限なし、残業も可能）

## 留学生の進路

### 進学と就職

| 進路 | 詳細 |
|------|------|
| **進学（多数派）** | ほとんどが専門学校を希望。大学進学は少数派 |
| **就職** | 技術・人文知識・国際ビザ（ギジンコク）or 特定技能ビザ |

### なぜ専門学校を選ぶのか

- 「学びたいことがあるから」ではなく「みんなが行っているからとりあえず行く」がほとんど
- 「国際ビジネス学科」のような専門学校は日本人学生を獲得できないため、外国人が中心
- 母国の留学紹介機関がビザについて正しく説明していない（特定技能ビザの存在を知らない機関もある）
- 特定技能ビザが2019年にできる前は、就職には現地4年制大学卒が必須だったため、専門学校に進学する習慣が残っている

::: info 留学生の背景
留学生は長く日本に在住したい（10年くらい働いて帰国結婚するパターンが多い）のに、なぜか最初に専門学校に進学する構造がある。これは現地での情報不足が主因。
:::

### 東京志向の理由

- 専門学校の多くが東京にある
- 東京はアルバイト代が高い（週28時間を無視して月給30万円の留学生もいる）
- 先輩から「東京は稼げる」と聞いている

## 就労ビザの種類

留学生が卒業後に就労する場合の代表的なビザ。

### 技術・人文知識・国際ビザ（ギジンコク）

| 項目 | 内容 |
|------|------|
| **要件** | 現地の4年制大学卒業が原則必須 |
| **家族滞在** | 認められる（妻・子供を日本に呼べる） |
| **主な仕事** | 派遣会社での通訳、ホテルでの通訳、監理組合での通訳 |
| **注意点** | 大学で学んだ内容と業務内容の関連性が求められる |

- 文系（経済・商学部・法学部等）の卒業者は比較的取得しやすい
- 日本の専門学校卒業（専門士）や日本の4年制大学卒業（学士）でも取得可能
- 実際に日本企業で働いている外国人は、日本語学校→日本の4年制大学を卒業しているケースがほとんど

### 特定技能ビザ

| 項目 | 内容 |
|------|------|
| **要件** | 対象分野の技能評価試験合格 + JLPT N4以上 |
| **家族滞在** | 認められない |
| **学歴要件** | なし（高卒でもOK） |

**対象分野**（16分野）:

| 分野 | 留学生の就職が多い分野 |
|------|----------------------|
| 介護 | |
| ビルクリーニング | |
| 素形材産業 | |
| 産業機械製造業 | |
| 電気・電子情報関連産業 | |
| 建設業 | |
| 造船・舶用工業 | |
| 自動車整備業 | |
| 航空業 | |
| 宿泊業 | ★ |
| 農業 | |
| 漁業 | |
| 飲食料品製造業 | ★ |
| 外食業 | ★ |
| 自動車運送業 | |

::: tip 特定技能の意義
特定技能ビザができたことは日本語学校にとって画期的。留学生に提示できる選択肢が増え、高卒でも卒業後すぐに就職可能になった。
:::

### その他のビザ

| ビザ | 内容 |
|------|------|
| **経営管理ビザ** | インドカレー店・中華料理店のオーナーなど。資本金500万円が目安 |
| **技能ビザ** | 主に料理人（中華料理・インドカレー等） |
