# データ — 制度・要件・試験

日本語教育機関に関する制度、設置要件、人的要件、試験情報をまとめたデータ集です。

## 学校ランク（適正校基準）

入管庁は日本語学校を過去の在籍管理実績に基づきランク付けし、ビザ審査の厳格度を区分しています。

| ランク | 名称 | 条件 | ビザ審査 |
|--------|------|------|----------|
| Class I | 適正校（優良） | 問題在籍者が在籍者の1%以下 | 書類簡素化。交付率が高い |
| Class II | 適正校（普通） | 問題在籍者が在籍者の3%以下 | 標準的な審査 |
| Class III | 適正校外（要注意） | 問題在籍者が在籍者の3%超〜5%以下 | 審査が厳しくなる |
| Class IV | 適正校外（問題校） | 問題在籍者が在籍者の5%超 | 交付率が大幅に低下。改善計画の提出を求められる |

::: tip 備考
適正校の判定は毎年行われ、過去1年間の実績で見直されます。一度「適正校外」になると、回復に最低1年かかります。
:::

## 問題在籍者の定義

入管庁が「問題在籍者」とみなす学生は、以下の5項目のいずれかに該当する者です。

| # | 定義 | 具体例 |
|---|------|--------|
| 1 | 不法残留者 | ビザの在留期限を超過して日本に滞在している者 |
| 2 | 不法就労者 | 資格外活動許可の範囲（週28時間）を超えて就労した者 |
| 3 | 在留資格取消者 | 虚偽申請等により在留資格を取り消された者 |
| 4 | 退去強制者 | 入管法違反により退去強制令書が発付された者 |
| 5 | 所在不明者 | 住居地を届け出ず、学校とも連絡が取れなくなった者 |

::: warning 注意
問題在籍者の比率は学校ランクに直結するため、在籍管理は日本語学校の最重要業務の一つです。
:::

## 設置要件

日本語教育機関の告示基準に定められた設置に関する主な要件です。

### 校舎・教室

| 項目 | 要件 |
|------|------|
| 教室面積 | 1教室あたり最低20平方メートル以上 |
| 1人あたり面積 | 学生1人あたり1.5平方メートル以上 |
| 教室の条件 | 採光・換気が十分であること |
| 校舎 | 教育に適した構造・設備を有すること |

### 授業時間

| 項目 | 要件 |
|------|------|
| 年間授業時間 | 760時間以上 |
| 1単位時間 | 45分以上 |
| 1日の授業時間 | 4単位時間以上 |
| 週あたり授業日数 | 5日以上 |

### クラスサイズ

| 項目 | 要件 |
|------|------|
| 1クラスの上限 | 20名以下 |
| 教員1人あたり | 1クラスに1名以上の専任教員を配置 |

## 人的要件

日本語教育機関に必要な人員配置と資格要件です。

| 役職 | 必要数 | 要件 |
|------|--------|------|
| 校長 | 1名 | 教育に関する識見を有し、教育機関の運営に必要な能力を有する者 |
| 教務主任 | 1名 | 日本語教育に関し5年以上の経験を有し、教務を統括する能力を有する者 |
| 専任教員 | 3名以上 | 学生40名につき1名以上。資格要件を満たすこと |
| 非常勤教員 | 必要数 | 資格要件を満たすこと |
| 生活指導担当 | 1名以上 | 学生の生活指導を行う者。学生の母語又は英語で対応できることが望ましい |
| 事務職員 | 必要数 | 学校運営に必要な事務を処理する者 |

## 教員の資格要件

日本語教育機関で教員として勤務するための資格要件は以下のいずれかを満たす必要があります。

| # | 資格要件 |
|---|----------|
| 1 | 大学（短期大学を除く）において日本語教育に関する主専攻（45単位以上）を修了し、卒業した者 |
| 2 | 大学（短期大学を除く）において日本語教育に関する副専攻（26単位以上）を修了し、卒業した者 |
| 3 | 日本語教育能力検定試験に合格した者 |
| 4 | 学士の学位を有し、かつ日本語教師養成講座（420時間以上）を修了した者 |
| 5 | その他、上記と同等以上の能力があると認められる者（大学院で日本語教育を専攻した者等） |

::: info 補足（2024年以降の新制度）
認定日本語教育機関への移行後は「登録日本語教員」の資格が必要になります。既存教員には経過措置（5年間）が設けられています。
:::

## 登録日本語教員制度（2024年〜）

2024年4月施行の日本語教育機関認定法により、日本語教員は国家資格「登録日本語教員」になりました。

### 資格取得ルート

| ルート | 内容 |
|--------|------|
| 試験ルート | 基礎試験合格 → 応用試験合格 → 実践研修修了 → 登録申請 |
| 養成機関ルート | 登録養成機関の課程修了（基礎試験免除）→ 応用試験合格 → 実践研修修了 → 登録申請 |

### 現職者の経過措置（6ルート）

「現職者」とは、2019年4月1日〜2029年3月31日の間に告示機関等で日本語教員として1年以上勤務した者です。

| ルート | 対象者 | 免除内容 | 必要な対応 | 期限 |
|--------|--------|----------|-----------|------|
| C | 現職者＋日本語教育能力検定試験合格（H15.4.1以降） | 基礎・応用・実践すべて免除 | 講習I＋講習修了認定試験 | 2033年3月末 |
| D-1 | 現職者＋学士以上＋平成12年報告対応の養成課程修了 | 基礎・実践免除 | 講習II＋講習修了認定試験 | 2029年3月末 |
| D-2 | 現職者＋学士以上＋告示基準対応の養成課程修了 | 基礎・実践免除 | 講習I＋講習II＋各認定試験 | 2029年3月末 |
| E-1 | 現職者＋日本語教育能力検定試験合格（S62〜H15.3.31） | 基礎・応用・実践すべて免除 | 講習I＋講習II＋各認定試験 | 2029年3月末 |
| E-2 | 現職者＋学士以上＋平成12年報告対応の養成課程修了 | 基礎・実践免除 | 講習II＋講習修了認定試験 | 2029年3月末 |
| F | 現職者で上記いずれにも非該当 | 実践研修のみ免除 | 基礎試験＋応用試験の合格 | 2029年3月末 |

::: warning 常南国際学院への影響
既存教員がどのルートに該当するかを早期に確認し、必要な講習受講・試験受験のスケジュールを立てる必要があります。特にルートD-2, E-1, E-2, Fは2029年3月末が期限です。
:::

## 主要な日本語能力試験

日本語学校の学生が受験する主な試験の一覧です。

| 試験名 | 正式名称 | 実施主体 | レベル | 実施回数 | 主な用途 |
|--------|----------|----------|--------|----------|----------|
| JLPT | 日本語能力試験 | 国際交流基金・日本国際教育支援協会 | N1〜N5（5段階） | 年2回（7月・12月） | 進学・就職の最も一般的な指標 |
| JFT-Basic | 国際交流基金日本語基礎テスト | 国際交流基金 | 合格/不合格 | 年6回程度（CBT） | 特定技能ビザの要件 |
| J-TEST | 実用日本語検定 | 日本語検定協会 | A-C級 / D-E級 / F-G級 | 年6回（奇数月） | JLPTの補完。中国圏で認知度が高い |
| NAT-TEST | 日本語NAT-TEST | 専門教育出版 | 1級〜5級（5段階） | 年6回（偶数月） | JLPTと同等の構成。ビザ申請の参考資料に使われる |

## 学生のJLPTレベル目安

日本語学校での学習期間とJLPTレベルの一般的な対応関係です。

| 学習期間 | 到達目安 | 説明 |
|----------|----------|------|
| 入学時 | N5相当またはそれ以下 | 基本的なひらがな・カタカナ・簡単な挨拶 |
| 6ヶ月 | N4相当 | 基本的な日常会話。簡単な文章の読み書き |
| 1年 | N3相当 | 日常的な場面で使われる日本語の理解 |
| 1年半 | N2相当 | 日常的な場面に加え、幅広い場面の日本語の理解。進学・就職に必要 |
| 2年 | N1相当 | 幅広い場面で使われる日本語の理解。大学院進学・専門職に必要 |

::: tip 備考
これはあくまで目安です。学生の出身国・母語・学習態度により大きく異なります。漢字圏（中国・台湾）の学生は非漢字圏（ベトナム・ネパール等）の学生より進度が早い傾向があります。
:::

## 告示校から認定日本語教育機関への移行

2024年の法律施行により、日本語教育機関は従来の「告示校」制度から「認定日本語教育機関」制度に移行中です。

| 項目 | 告示校（旧制度） | 認定日本語教育機関（新制度） |
|------|-----------------|--------------------------|
| 所管 | 法務省（入管庁） | 文部科学省 |
| 根拠 | 出入国管理及び難民認定法の告示 | 日本語教育機関認定法（2024年施行） |
| 教員資格 | 告示基準の要件 | 登録日本語教員（国家資格） |
| 認定基準 | 告示基準 | 認定基準（文科省） |
| 移行期限 | — | 2029年3月末（経過措置5年） |
| 報告先 | 入管庁 | 文科省＋入管庁（ビザ関連は引き続き入管庁） |

::: warning 移行期の留意点
- 2029年3月末までに認定を受けなければ「留学」ビザの受け入れができなくなる
- 既存教員は経過措置期間内に「登録日本語教員」の資格を取得する必要がある
- 認定申請には教育課程の整備・自己点検評価の実施等が求められる
:::

### 認定申請の実績（参考）

| 公表時期 | 申請件数 | 認定件数 | 認定率 |
|---------|---------|---------|--------|
| 2024年10月 | 72件 | 22件 | 約31% |
| 2025年3月 | 48件 | 19件 | 約40% |
| 2025年10月 | 74件 | 23件 | 約31% |

認定申請は年2回（4月申請→10月結果、10月申請→翌4月結果）。2025年11月時点で累計64機関が認定済み。

### 認定日本語教育機関の課程類型

| 類型 | 説明 |
|------|------|
| 留学のための課程 | 旧法務省告示校に相当。常南国際学院はこの類型 |
| 就労のための課程 | 就労目的の日本語教育 |
| 生活のための課程 | 地域の生活者向け日本語教育 |

### 認定基準で新たに求められること

| 項目 | 内容 |
|------|------|
| 教育課程 | CAN-DO記述文による到達目標設定。CEFRレベルでの段階的目標 |
| 自己点検評価 | 10項目（教育理念・運営・情報公開・入学選考・教育活動・教職員養成・学生支援・施設設備・地域貢献・財務）を年1回以上実施・公表 |
| 情報公表 | 教育目的・学習成果・評価実施状況等をWebサイト等で公表する義務 |
| 認定の更新 | 5年ごとの更新制 |

## 告示基準の全体構成（第1条 第1号〜第48号）

告示基準は入管庁が定めた日本語教育機関の要件です。第1条に48号、第2条（抹消基準）に8号があります。

### 主要な号の分類

| 号 | 分類 | 概要 |
|----|------|------|
| 第1号〜第2号 | 名称・学則 | 適当な名称であること、学則の策定 |
| 第3号〜第6号 | 設置者 | 設置者の適格性・欠格事由・経営基盤 |
| 第7号〜第14号 | 校長・教員 | 校長設置・主任教員・教員配置数・資格要件・担当時間上限 |
| 第15号〜第17号 | 事務・生活指導 | 事務統括職員・生活指導体制 |
| 第18号 | 自己点検 | 年1回以上の自己点検・評価 |
| 第19号〜第27号 | 施設・設備 | 校地校舎の要件・面積基準・教室環境 |
| 第28号〜第30号 | 教育課程 | クラスサイズ20名以下・年間760時間以上 |
| 第31号〜第34号 | 入学選考 | 情報提供・経費支弁確認・選考記録保存 |
| 第35号〜第40号 | 在籍管理 | 出席率報告・8割未満指導・5割未満報告・アルバイト管理 |
| 第41号 | 禁止行為 | 人権侵害・法令違反の禁止 |
| 第42号〜第43号 | 変更報告 | 変更の届出・事業計画書 |
| 第44号 | 学習成果 | 課程修了者の7割以上が進学等であること |
| 第45号 | 定期点検 | 毎年4月1日時点の適合性を6月末まで報告 |
| 第46号〜第48号 | その他 | 記録保存・入管協力・文科大臣意見聴取 |

### 告示抹消の基準（第2条）

以下のいずれかに該当すると告示から抹消され、留学生の受入れができなくなります。

| 号 | 抹消事由 |
|----|----------|
| 第1号 | 学則・誓約書の遵守義務に違反 |
| 第2号 | 第1条の要件に不適合 |
| 第3号 | 全学生の6か月出席率平均が**7割を下回った** |
| 第4号 | 一暦年の入学者の**3割以上**が不法残留 |
| 第5号 | 在籍管理不適正の通知を**3年連続**で受けた |
| 第6号 | 課程修了者の学習成果が**3年連続で7割を下回った** |
| 第7号 | 教育活動を行っていない学生が相当数存在 |
| 第8号 | 学生に対する人権侵害・法令違反の教唆 |
