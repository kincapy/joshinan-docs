# 日本語教育機関（業界全体）

日本語学校の業界知識、法的要件、制度の基盤をまとめたカテゴリです。

::: tip この文書の趣旨
日本語学校がどういう組織かわからない人が、これを読んで日本語学校とはどういう機関でどういう仕組みなのかを理解し、初心者でも日本語学校の運営を理解できることを目的としています。
:::

## 日本語学校とは何か（総論）

### 基本的な定義

日本語学校とは単なる語学学校（例：駅前留学のような学校）ではなく、海外から外国人を在留資格「留学」ビザで招聘できる権利を有している学校を指す。

- **準拠法**: 法務省の「日本語教育機関の告示基準」
- **告示校**: 上記の権利を有している学校の呼称
- **現在の状況**: 2024年5月に「日本語教育の適正かつ確実な実施を図るための日本語教育機関の認定等に関する法律」が制定され、「認定日本語教育機関」制度が誕生。現在は告示校から認定日本語教育機関への移行期間中

### 認定法が制定された背景

- 技能実習生や特定技能などの海外労働者が増加する中、日本語教育の質を担保する目的で制定
- 告示校時代は法律ではなく「告示」に基づく機関であり、法的立場がそこまで強くなかった
- 新法律により法的な立場が強化された
- 日本語教師も「登録日本語教員」という国家資格に変更（詳細は [09 教職員管理](/01-domain-knowledge/09-staff-management/) 参照）

### 日本語教育機関の役割（総論）

日本語教育機関の役割は大きく分けて以下の3つ。

1. **日本語教育**: 留学生に対して日本語を教育すること
2. **生活支援**: 留学生が円滑に日本の生活に順応できるようサポートすること
3. **在籍管理**: 留学生にきちんと入管法のルールを理解させ従わせること

::: info 実態
日本語学校はただ日本語を教えるだけでは足りず、アルバイトの紹介、ゴミ捨てなどの生活指導、進学先・就職先の支援、病院への付き添いなど、雑多な作業も数多く行っているのが実態。
:::

### 学校のランク

入管（出入国在留管理庁）は教育機関を評価し、3つのランクに分類している。

| ランク | 名称 | メリット |
|--------|------|----------|
| 適正校（クラスI） | 在籍管理優良校 | 募集書類・ビザ更新書類が簡素化。事業運営上非常に有利 |
| 適正校（クラスII） | 適正校 | 標準的な審査 |
| 慎重審査対象校 | 在籍管理不適正校・新規校 | 審査が厳しい。3年連続で許可取消事案となる |

**適正校の要件**（入管より引用）:

> 前年1月末の在籍者数に占める問題在籍者の割合（問題在籍率）が5%以下であること。ただし、前年1月末の在籍者数が19人以下である場合は、問題在籍者が1人を超えないこと。

**問題在籍者の定義**:

1. 不法残留した者
2. 在留期間更新許可申請が不許可となった者
3. 在留資格を取り消された者
4. 資格外活動の許可を取り消された者
5. 退去強制令書が発付された者

::: warning クラスIの重要性
クラスIになると交付率の不安定性に左右されにくくなり、事業運営上の安定性が増す。日本語学校を運営する上での最重要指標の一つ。
:::

### 法務省から文科省への転換点

現在、日本語教育業界は大きな転換点を迎えている。

- **法的変化**: 告示レベルの制度から法律を準拠にした制度へ
- **教育方法の変化**: 従来の「文型積み上げ方式」から「CAN-DO型式」（実際の生活での会話を重視する型式）へ
  - 文型積み上げ方式 = 文法・読解重視（例：TOEIC型）→ 読み書きはできるが喋れない
  - CAN-DO型式 = 会話重視（例：IELTS型）→ 実践的なコミュニケーション能力
- **現場への影響**: 代表的な教科書「みんなの日本語」はCAN-DO型式には不向き。認定日本語教育機関への移行が難航している学校も多い

## 設置要件

### 校舎・施設

| 項目 | 要件 |
|------|------|
| 校舎面積 | 115平方メートル以上 |
| 1人あたり面積 | 同時授業の学生1人あたり2.3平方メートル以上 |
| 校地・校舎の所有 | 原則として設置者の所有。公共機関からの賃借で20年以上安定使用可能な場合は例外 |
| 教室設備 | 採光・換気・冷暖房設備・机・椅子・ホワイトボード等 |
| 附属施設 | 職員室・事務室・図書室・学生相談室・保健室など |

### 教育課程・学習時間

| 項目 | 要件 |
|------|------|
| 年間授業時間 | 760時間以上（実際は800時間が多い） |
| 週あたり授業時間 | 20時間以上 |
| 年間授業週数 | 38週以上 |
| 授業形態 | 半日授業（午前or午後の4時間）。年間190〜200日 |

### 運営主体

| 項目 | 要件 |
|------|------|
| 法人格 | 株式会社・学校法人・公益法人など |
| 経営基盤 | 財務状況・収支予測等により健全かつ継続的な運営が可能であること |

### 学生支援・管理体制

| 項目 | 要件 |
|------|------|
| 生活支援 | 入国後の生活指導・健康管理・アルバイト指導体制 |
| 記録管理 | 出席管理・学習状況・生活状況の記録を適切に管理 |
| 緊急対応 | 緊急時の対応体制が明確に構築されていること |
| 住居支援 | 入学時点で住居確保の見込みがあること。通学可能な範囲で学生寮または借上住宅の提供・紹介 |

### クラス定員

| 項目 | 要件 |
|------|------|
| 1クラスの上限 | 20名まで |
| 教室・教員とのバランス | 校舎面積・教室数が受入予定学生数と整合し、過密状態を避けること |

## 主要な役割（各論）

### 言語教育機関としての役割

- 年間授業時間は最低760時間（実際は800時間が多い）
- 半日授業（午前or午後4時間）で、年間190〜200日
- 夏休み・春休みなどの長期休みあり
- 基本指標は**JLPT**（日本語能力試験、年2回/7月・12月）

**留学生のJLPTレベル目安**:

| 時期 | レベル | 評価 |
|------|--------|------|
| 入国時 | N5前後 | — |
| 卒業時（2年後） | N3合格 | 優秀 |
| 卒業時（2年後） | N2以上 | かなり優秀 |
| 卒業時（2年後） | N4以下 | かなりのケースで存在 |

### 生活支援拠点としての役割

日本語学校が行う生活支援業務は以下の通り。

| 業務 | 内容 |
|------|------|
| **寮の準備** | 留学生は入国前に不動産契約が困難なため、学校が物件を契約。電気・ガス・水道の契約、二段ベッド・机・冷蔵庫・洗濯機・Wi-Fi・コンロ・食器・自転車などの備品整備 |
| **入国時手続き** | 市役所の転入手続き、国民健康保険の加入、国民年金の加入・免除申請、ゆうちょ銀行の口座作成 |
| **アルバイト紹介** | 「資格外活動」許可で週28時間まで就労可能。学費・生活費の捻出に不可欠であり、留学生生活の生命線 |
| **生活指導** | ゴミの分別、自転車の乗り方・交通ルール、騒音防止、土足禁止など。教えないと近隣クレームの原因に |

### その他の役割

- **文化交流**: 日本文化理解のイベント実施、地域ボランティア活動への参加
- **進路指導**: 専門学校への進学支援、技術・人文知識・国際ビザや特定技能ビザでの就職支援

## 入管報告

入管報告の詳細は [11 入管報告・届出](/01-domain-knowledge/11-immigration-report/) を参照。

### 定期報告

| 報告書 | 提出時期 | 内容 |
|--------|----------|------|
| 在籍状況等報告書 | 年2回（4月・10月） | 全学生の在籍状況・出席率 |
| 出席率報告書 | 年2回（4月・10月） | 各生徒の過去6ヶ月間の出席率を個別報告 |
| 自己点検・評価報告書 | 年1回（6月末まで） | 教育運営の自己点検、失踪率、進学率、教職員体制 |
| 卒業者報告書 | 年1回（卒業後） | 課程修了者の一覧 |

### 随時報告

| 報告書 | 提出時期 | 内容 |
|--------|----------|------|
| 退学・除籍・失踪者の届出 | 該当者発生時速やかに | 退学理由、帰国・転校・失踪の詳細 |
| 出席率50%未満の報告 | 随時 | 出席率50%未満の者の状況と指導内容 |
| 入学者名簿 | 学期開始時（4月・10月） | 入国・入学を確認した学生リスト |
| 受入終了届 | 該当時 | 新規受入停止の通知 |
| 在籍終了届 | 卒業・退学時 | 在籍しなくなったことの報告 |

### ビザ更新

- **タイミング**: 適正校の場合、入国時1年3ヶ月のビザ → 4月入学なら翌年7月に更新（在学中1回）
- **新規校の場合**: 入国時6ヶ月ビザのため、更新タイミングがずれる

**提出書類**:

| 区分 | 書類 |
|------|------|
| 学校準備 | 在学証明書、成績証明書、出席状況証明書（過去6ヶ月）、所属機関等作成用1・2 |
| 本人準備 | 在留期間更新許可申請書（写真貼付）、証明写真、パスポート、在留カード、預金通帳の写し、源泉徴収票・給与明細、仕送り・送金証明書 |
| その他 | 提出書類チェックリスト、収入印紙（6,000円、オンライン申請は5,500円） |

::: warning ビザ更新が重要な理由
ビザ更新に失敗した学生は問題在籍者にカウントされ、適正校の要件に影響する。主な不許可理由は以下の2つ:

1. **出席率の低さ**: 最低85%以上がないと失敗する場合がある
2. **アルバイトのオーバーワーク**: 週28時間超の就労が通帳や源泉徴収で発覚した場合

学校が把握した場合、ビザ更新手続きをせず帰国させることで問題在籍者数にカウントさせない手法もある。
:::

### 入管報告の業務工数

- 入管報告自体はそこまで工数はかからない
- 主に事務局長や校長が担う業務領域
- ただし入管報告だけでは人的効率が低いため、他の業務を兼務する必要がある
- 書類作成自体も日本語学校向けシステムを使えばパート・アルバイトでも作成可能なレベル
