# 日本語教育機関（業界全体）

日本語学校の業界知識、法的要件、制度の基盤をまとめたカテゴリです。

## 概要

日本語学校は単なる語学学校ではなく、在留資格「留学」ビザで外国人を招聘できる権利を有する教育機関です。

## 準拠法

- 日本語教育機関の告示基準（法務省）
- 2024年5月制定「日本語教育の適正かつ確実な実施を図るための日本語教育機関の認定等に関する法律」
- 現在、告示校から認定日本語教育機関（文科省）への移行期
