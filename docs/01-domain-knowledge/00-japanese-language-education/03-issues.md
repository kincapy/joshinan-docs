# 課題

日本語教育機関（業界全体）に関する課題をまとめています。

## 課題1: 告示校から認定校への移行対応

### 課題

2024年施行の日本語教育機関認定法により、すべての日本語学校は2029年3月末までに「認定日本語教育機関」への移行を完了しなければならない。移行が完了しなければ「留学」ビザでの学生受け入れができなくなる。

### 背景

- 従来の「告示校」制度は法務省（入管庁）が所管していたが、新制度では文部科学省が所管する
- 教員資格が「登録日本語教員」（国家資格）に一本化される
- 既存教員には5年間の経過措置があるが、その間に資格取得が必要
- 認定申請には教育課程の整備、自己点検評価の実施、教員の資格対応など多岐にわたる準備が必要
- 小規模校にとっては事務負担が大きく、対応が遅れるリスクがある

### ポイント

- **期限が明確**：2029年3月末が移行期限。逆算してスケジュールを立てる必要がある
- **教員の資格取得**：既存教員の登録日本語教員への移行支援（受験費用の補助、研修時間の確保）が急務
- **二重報告体制**：移行期間中は文科省と入管庁の両方に対応が必要になる可能性がある
- **情報収集**：文科省からの通達・ガイドラインを継続的にウォッチする体制が必要
- **自己点検評価**：認定基準に合わせた自己点検評価の仕組みを整備する必要がある

## 課題2: ビザ交付率の不安定性

### 課題

在留資格認定証明書（COE）の交付率は、入管庁の審査方針・国際情勢・学校のランクにより大きく変動し、学校経営の安定性を脅かす要因となっている。

### 背景

- COEの交付率は国籍・地域ごとに異なり、年度や時期によっても大きく変動する
- ベトナム・ネパール・スリランカなど一部の国籍では不交付率が高い傾向がある
- 学校のランク（適正校基準のClass I〜IV）により審査の厳格度が異なる
- 不交付となった場合、学生募集のやり直し・エージェントとの調整・収入計画の見直しが必要
- 入管庁の審査基準は非公開であり、不交付理由の詳細が開示されないことが多い

### ポイント

- **経営リスク**：交付率の低下は直接的に入学者数・学費収入の減少につながる
- **国籍の分散**：特定国籍に偏ると交付率変動のリスクが集中するため、募集国の分散が重要
- **適正校維持**：問題在籍者の比率を低く保ち、Class I を維持することがビザ交付率の安定に直結する
- **エージェント管理**：送出機関の質がビザ申請書類の質に影響するため、エージェントの選定・管理が重要
- **書類品質**：申請書類の不備・矛盾を事前にチェックする体制を整備し、不交付リスクを最小化する
