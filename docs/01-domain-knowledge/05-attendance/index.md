# 出席管理

法定要件に基づく出席率の管理を行うカテゴリです。出席率は入管への報告義務があり、適正校の維持に直結します。

## 運用システム

出席管理は **RINGUAL** で行います。クラス単位でフィルタし、各学生の各時限に対して出欠ステータスを入力します。

### 出欠入力の流れ

1. RINGUALの「出欠入力」画面を開く
2. クラス（A1〜A4、P1〜P3）でフィルタ
3. 各学生の各時限に対して出欠ステータスを選択
4. 1日分の出欠を一括入力する機能もあり

### 出欠ステータス（6種）

| コード | ステータス | 出席率への影響 |
|--------|-----------|--------------|
| 0 | （未入力） | — |
| 1 | 出席 | 出席としてカウント |
| 2 | 欠席 | 欠席としてカウント |
| 3 | 遅刻 | 遅刻4回 = 欠席1回（設定で変更可能） |
| 4 | 早退 | — |
| 5 | 公欠 | **出席扱い** |
| 6 | 出停 | 出席停止（感染症等） |

::: tip 遅刻の換算
遅刻4回で欠席1回に換算されます。この換算率はRINGUALの設定で変更可能です。
:::

### 出欠データのエクスポート

- 週単位（1〜5週分）でExcelエクスポートが可能
- 性別フィルターも利用可能（男性 / 女性）
