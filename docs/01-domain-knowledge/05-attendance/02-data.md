# データ — 出席率の基準

## 出席率の基準値

| 基準 | 値 | 影響 |
|------|-----|------|
| 告示基準の最低ライン | 出席率7割 | 下回ると告示抹消の事由 |
| 指導記録が必要なライン | 出席率8割未満 | 指導記録の作成・保存が必須 |
| 入管報告ライン | 出席率5割未満 | 該当月の翌月末までに報告義務（告示基準第39号） |
| 適正校維持の参考値 | 出席率8割以上 | クラスⅠ維持の目安 |

## 報告対象期間

| 期間 | 対象 | 報告期限 |
|------|------|----------|
| 前期 | 4月1日〜9月30日 | 12月末まで |
| 後期 | 10月1日〜翌年3月31日 | 6月末まで |
