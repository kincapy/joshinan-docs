## 指導記録

| フィールド | 値の例・選択肢 |
|-----------|--------------|
| 投稿日 | 日付 |
| 編集日 | 日付（編集済みマーク） |
| タイトル | テキスト |
| 本文・コメント | テキスト（長文可） |
| 投稿者 | 教職員名 |

| 公開/限定 | 公開 / 限定 |
| タグ | 指導 / 報告 / 進路（複数選択可） |

::: tip 指導記録の活用
指導記録は **定期面談の記録** として活用されている。面談では以下を記録：
1. 生活状況（寮の掃除、生活リズム）
2. 学習状況（授業理解度、自宅学習、JLPT受験予定）
3. アルバイト（勤務先、時間、時給）
4. 進路（進学先/就職先の検討状況）
5. 出席率と注意事項
:::

## 学籍番号の体系

```
YYMM + 連番3桁
```

| 例 | 意味 |
|----|------|
| `2404001` | 2024年4月入学の1番目 |
| `2510001` | 2025年10月入学の1番目 |
| `2601001` | 2026年1月入学の1番目 |
| `9999999` | テスト用（推定） |
