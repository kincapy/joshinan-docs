# 外部システム連携

Ringual、Box、Google Driveなどの外部システムとの連携を管理するカテゴリです。

## データソースと連携方式

| データソース | 役割 | 連携方式 |
|-------------|------|----------|
| Ringual | 学生情報のオーソリティ | 手動エクスポート → Excel取り込み |
| Box | ファイルストレージ（本番データ） | MCP経由（現在）→ API（将来） |
| Google Drive | 家賃・水光熱費データ | MCP経由 |
| りそな銀行 | 海外送金受取 | CSV取り込み（予定） |
| 足利銀行 | 国内振込 | CSV取り込み（予定） |
| 暮らしてぷこ（TEPCO） | 水光熱費管理 | 手動確認 |
| 入管電子申請システム | 在籍・出席状況報告 | 手動操作（マニュアル化済み） |

## Ringual の位置づけ

常南国際学院の学生管理における **オーソリティ（正のデータ）** はRingual。以下の3つが主な活用領域：

1. **学生データベース** — 入管申請に必要な全フィールドを網羅。COE申請書や在留期間更新許可申請書を直接出力可能
2. **クラス設定 → 出欠管理** — クラス単位で出欠を管理し、出席率を自動計算
3. **指導記録** — 定期面談や生活指導の記録をタグ付きで蓄積
