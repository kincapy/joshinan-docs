# RINGUAL — 出欠・成績・時間割・クラス管理

Playwrightによるクロール結果に基づく、RINGUALの運用系画面の詳細。

## 出欠入力（student_attendance.php）

### 出欠ステータス

| コード | ステータス | 備考 |
|--------|-----------|------|
| 0 | （未入力） | — |
| 1 | 出席 | — |
| 2 | 欠席 | — |
| 3 | 遅刻 | 遅刻4回 = 欠席1回（設定で変更可能） |
| 4 | 早退 | — |
| 5 | 公欠 | 出席扱い |
| 6 | 出停 | 出席停止（感染症等） |

### 操作方法

- クラス別にフィルタ（ClassList）して学生一覧を表示
- 各学生の各時限に対して出欠ステータスを選択
- 1日分の出欠を一括入力する機能あり（modalMultiInputValue）
- 週単位（1〜5週分）でExcelエクスポート可能

### フィルター

| フィルター | 選択肢 |
|-----------|--------|
| クラス | A1〜A4クラス（午前）、P1〜P3クラス（午後） |
| 性別 | 男性 / 女性 |
| エクスポート週数 | 1〜5週分 |

## 成績入力（student_grade.php）

### 試験区分（TestRelationPeriod）

| コード | 区分 |
|--------|------|
| 1 | 1年_前期期末試験 |
| 2 | 1年_後期期末試験 |
| 3 | 2年_前期期末試験 |
| 4 | 2年_後期期末試験 |
| 0 | 最新学業成績 |

### 合否判定

| コード | 判定 |
|--------|------|
| 1 | 合格 |
| 2 | 不合格 |
| 3 | 退場 |
| 4 | 欠席 |

### フィルター

| フィルター | 用途 |
|-----------|------|
| 試験対象クラス | クラス別の成績表示 |
| 学科コース | コース別フィルタ |
| 入学年度 | 年度別フィルタ（2024 / 2025 / 2026） |
| 合否 | 合格 / 不合格 / 退場 / 欠席 |

## 時間割（timetable.php）

### 構造

時間割は **クラス × 曜日 × 時限** の三次元で管理される。

| 項目 | 内容 |
|------|------|
| クラス | A1〜A4、P1〜P3（年度ごと） |
| 曜日 | 月〜土（日曜含む7曜日） |
| 時限 | 1限〜複数時限 |
| 科目 | 文法、N3文法、みん日中級1 等 |

### ID体系

時間割セルのID: `sel-tt-{曜日番号}_{時限番号}`

| 曜日番号 | 曜日 |
|---------|------|
| 0 | 日 |
| 1 | 月 |
| 2 | 火 |
| 3 | 水 |
| 4 | 木 |
| 5 | 金 |
| 6 | 土 |

## クラス登録（class_regist.php）

### クラス設定項目

| フィールド | 内容 |
|-----------|------|
| クラス名 | 例: A1クラス(2025年度） |
| 印刷用の名称 | 証明書等に印刷される名前 |
| レベル | N2, N3, N4, N5 等 |
| 有効期間（開始/終了） | クラスの開講期間 |
| サブクラスフラグ | 学生が一時的に所属するクラス |
| 参照レベル | A1, A2, B1, B2, C1, C2（CEFR対応） |

### クラス体系（2025年度の例）

| クラス | 時間帯 | レベル |
|--------|--------|--------|
| A1クラス | 午前 | N2 |
| A2クラス | 午前 | N3 |
| A3クラス | 午前 | N4 |
| A4クラス | 午前 | N5 |
| P1クラス | 午後 | — |
| P2クラス | 午後 | — |
| P3クラス | 午後 | — |

### 表示フィルター

| フィルター | 選択肢 |
|-----------|--------|
| 表示条件 | 開講中 / 開講予定 / 期間内全て |
| 並び順 | クラス名 / レベル / 開始日（昇順/降順） |
| クラスタイプ | すべて / サブクラスなし / サブクラスのみ |
