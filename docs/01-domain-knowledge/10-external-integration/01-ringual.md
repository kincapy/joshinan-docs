# RINGUAL — 学生管理システム

## 概要

RINGUAL JAPON は、日本語教育機関向けの学生管理SaaS。開発元は株式会社日東システムテクノロジーズ。

常南国際学院では **学生データベース**、**出欠管理**、**指導記録** の3つを主に活用している。

| 項目 | 内容 |
|------|------|
| URL | `https://www.ringual.jp/system/6517ai7/` |
| 開発元 | NITTO SYSTEM TECHNOLOGIES CO., LTD. |
| プラン | フリープラン |
| ポイント制 | 93 / 90（期間：3/1〜9/30） |

## よく使う機能・使いにくい機能

### 使える機能

| 機能 | 評価 | 理由 |
|------|------|------|
| 学生データベース | **有能** | 入管申請に必要な全フィールドを網羅。COE申請書・在留期間更新許可申請書を直接出力可能 |
| クラス設定 → 出欠管理 | **比較的使える** | クラス単位で出欠を管理し、出席率を自動計算。遅刻4回＝欠席1回の換算にも対応 |
| 指導記録 | **結構使える** | タグ（指導/報告/進路）で分類。投稿者・日付・重要マークで検索可能。面談内容の蓄積に有効 |

### 使いにくい点

- UI が古く直感的でない
- 各画面間の遷移が遅い（ページ遷移がフォームPOST方式）
- Excel エクスポート時にカラム順が不安定なことがある
- 検索・フィルタの操作性が悪い

## 学生カルテのタブ構成

学生カルテ（`student_karte.php`）は以下のタブで構成される。

| タブ | 内容 |
|------|------|
| 基本情報 | 学籍番号、氏名、国籍、住所、旅券、入国目的、COE申請等 |
| 学生情報 | 学歴、経歴、日本語能力（JLPT等）、日本語教育機関の履歴 |
| 在留情報 | 在留カード、在留資格、期間、更新申請 |
| 保護者情報 | 在日親族（最大6名）の登録 |
| 保証人 | 保証人・監護人・代理人 |
| 支弁者 | 経費支弁方法、支弁者情報、奨学金 |
| 勤務先・資格外活動 | 勤務先（最大20件）、業種、時給、稼働時間 |
| 卒業後・進路 | 進路区分、進学先、就職先 |
| 学納金 | 請求・入金の管理 |
| 申請書・許可証 | 入管申請書類の出力 |
| ユーザーフォーム | カスタムフォーム |

## 出力可能な入管申請書類

RINGUALから直接出力できる入管関連書類：

| 書類名 | 用途 |
|--------|------|
| 在留資格認定証明書交付申請書 | 新規入国の申請 |
| 在留期間更新許可申請書 | ビザの延長申請 |
| 在留資格変更許可申請書 | 在留資格の変更 |
| 資格外活動許可申請書 | アルバイト許可の申請 |
| 申請者リスト | 一括申請用のリスト |
| 活動機関に関する届出（移籍/離脱） | 転校・卒業時の届出 |
| 中長期在留者の受入れに関する届出 | 入学・卒業時の届出 |
| 各入管向け別添書類 | 東京・札幌・大阪・仙台・名古屋の各入管固有の書式 |

## システム設定の構成

RINGUALのシステム設定（`setting.php`）には以下のメニューがある：

| 設定項目 | 内容 |
|---------|------|
| 学校情報 | 学校の基本情報 |
| 学科コース | コースマスタの管理 |
| 講師・職員 | 教職員アカウントの管理 |
| 遅刻・早退・出席率計算 | 出席率の計算ルール設定 |
| 授業時限数 | 1日あたりの時限数設定 |
| 学業成績設定 | 試験区分・評価基準の設定 |
| 自由項目 | カスタムフィールドの追加 |
| 行事・授業予定 | 学校行事カレンダー |
| 学生ポータルパスワード発行 | 学生向けポータルの設定 |
| 入学前学生ステータス詳細 | ステータスの細分類設定 |
| エージェント管理 | 送出機関マスタの管理 |
| 学納金メニュー | 費目・テンプレートの設定 |
| 管理者メニュー | 管理者権限の設定 |
