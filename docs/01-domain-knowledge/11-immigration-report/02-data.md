# データ — 入管届出の根拠法令・様式

## 根拠法令の一覧

日本語教育機関に関わる入管関連の法令・基準の体系です。

| 法令・基準 | 所管 | 概要 |
|-----------|------|------|
| 出入国管理及び難民認定法（入管法） | 法務省 | 在留資格制度、在留管理の基本法 |
| 入管法第19条の16 | 法務省 | 中長期在留者本人による所属機関変更届出（義務・罰則あり） |
| 入管法第19条の17 | 法務省 | 受入れ機関による受入れ状況届出（努力義務・罰則なし） |
| 日本語教育機関の告示基準 | 入管庁 | 告示校として満たすべき48号の要件 |
| 日本語教育機関の告示基準解釈指針 | 入管庁 | 告示基準の詳細な解釈・運用方針 |
| 日本語教育機関認定法 | 文科省 | 認定日本語教育機関制度の根拠法（2024年4月施行） |
| 認定法 施行規則 | 文科省 | 認定基準・報告義務の詳細 |

## 入管法第19条の17 の詳細

| 項目 | 内容 |
|------|------|
| 条文の趣旨 | 中長期在留者を受け入れている機関は、受入れ状況を届け出るよう努めなければならない |
| 法的性質 | 努力義務（罰則なし）。ただし告示基準では事実上の義務 |
| 対象 | 「留学」の在留資格で在留する学生を受入れている日本語教育機関 |
| 届出事項 | 氏名、生年月日、性別、国籍、住居地、在留カード番号、受入れの開始・終了 |
| 届出期限 | 事由発生から14日以内 |

### 届出方法

| 方法 | 詳細 |
|------|------|
| オンライン（推奨） | 電子届出システム。24時間365日。留学生は1人でもExcelファイルでの届出が必要 |
| 窓口 | 管轄の地方出入国在留管理局。職員の身分証明書を提示 |
| 郵送 | 届出書に身分証明書の写しを同封。封筒に朱書きで「届出書在中」 |

## 適正校制度の詳細

### 判定基準

| 項目 | 内容 |
|------|------|
| 分子 | 前年1月1日〜12月31日の問題在籍者の合計数 |
| 分母 | 当年1月31日時点の在籍者数 |
| 適正校 | 問題在籍率5%以下（在籍者19人以下の場合は問題在籍者1人以内） |

### クラス分けと影響

| 区分 | 基準 | 影響 |
|------|------|------|
| 適正校（クラスI） | 3年連続で1%以下 | 書類大幅簡素化、在留期間2年、定員増可能、定期点検3年ごと |
| 適正校（クラスII） | 5%以下 | 書類簡素化、在留期間2年 |
| 非適正校 | 5%超 | 書類増加、在留期間6か月、審査厳格化 |

### 問題在籍者の定義（令和元年改定後）

| # | 定義 |
|---|------|
| 1 | 不法残留した者 |
| 2 | 在留期間更新許可申請が不許可となった者 |
| 3 | 在留資格を取り消された者 |
| 4 | 資格外活動の許可を取り消された者 |
| 5 | 退去強制令書が発付された者 |

## 資格外活動許可の管理

| 項目 | 内容 |
|------|------|
| 制度 | 「留学」は就労不可の在留資格。資格外活動許可でアルバイト可能 |
| 制限時間 | 週28時間以内（連続する7日間で計算、複数アルバイト合算） |
| 長期休業期間 | 1日8時間以内まで可 |
| 禁止 | 風俗営業店舗等での就労 |
| 学校の義務 | アルバイト先の届出を求め、記録を在籍終了後1年間保存（告示基準第40号） |
| 5割未満の場合 | 出席率5割未満の学生はアルバイト先名称も併せて入管に報告 |

### 違反時のリスク

| 対象 | リスク |
|------|--------|
| 学生本人 | 退去強制、在留期間更新の不許可 |
| 学校 | 在籍管理不適正→適正校認定に影響 |
| 雇用先企業 | 不法就労助長罪 |

## 令和元年（2019年）告示基準改定の要点

| 項目 | 旧基準 | 新基準 |
|------|--------|--------|
| 適正校判断 | 不法残留者のみ（3%以内） | 不法残留＋不許可＋取消＋資格外活動取消（5%以内） |
| 出席率（抹消基準） | 5割以下で抹消 | **7割以下**で抹消 |
| オーバーステイ（抹消基準） | 半数以上で抹消 | **3割以上**で抹消 |
| 学習成果基準 | なし（新設） | 進学等が修了者の**7割以上**（3年連続未達で抹消） |
| 出席率報告 | — | 6か月ごとに年2回報告義務化 |
| 定期点検報告 | — | 毎年4月1日時点の適合性を6月末まで報告 |
