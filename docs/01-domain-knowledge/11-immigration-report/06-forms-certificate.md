## 長期休暇証明書

### 概要

| 項目 | 内容 |
|------|------|
| 用途 | 長期休暇中に資格外活動の時間延長（1日8時間以内）を行う際の証明 |
| 根拠 | 資格外活動許可の条件（入管法第19条第2項） |
| 発行者 | 日本語教育機関（学校長名で発行） |
| 提出先 | アルバイト先企業（求めに応じて提示） |

::: warning 長期休暇中の就労制限
通常は週28時間以内ですが、学校が定める長期休業期間中は1日8時間以内まで拡大されます。この証明書がないとアルバイト先が休暇期間を確認できません。
:::

### テンプレート構造（差し込みフィールド）

長期休暇証明書は定型テンプレートに学生情報と休暇期間を差し込んで出力します。

| フィールド | データ型 | 説明 |
|-----------|---------|------|
| 学校名 | 固定値 | 常南国際学院 |
| 学校所在地 | 固定値 | 茨城県つくば市榎戸433-2 |
| 学校電話番号 | 固定値 | 029-875-8828 |
| 学校長名 | テキスト | 校長の氏名（公開情報） |
| 発行日 | 日付 | 証明書の発行日 |
| 学生氏名 | テキスト | 対象学生の氏名 |
| 国籍 | テキスト | 学生の国籍 |
| 生年月日 | 日付 | 学生の生年月日 |
| 在留カード番号 | テキスト | 学生の在留カード番号 |
| 所属コース | テキスト | 在籍しているコース名 |
| 休暇名称 | テキスト | GW休暇 / 夏季休暇 / 冬季休暇 / 春季休暇 |
| 休暇開始日 | 日付 | 長期休暇の開始日 |
| 休暇終了日 | 日付 | 長期休暇の終了日 |
