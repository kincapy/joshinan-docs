# 帳票サンプル — ビザ申請

RINGUALから出力されるビザ関連申請書の構造とサンプルです。個人情報はダミーに置換しています。

- [業務フロー](./01-work-flow.md) — COE申請・在留期間更新の流れ
- [データ](./02-data.md) — 根拠法令・適正校制度

---

## 在留資格認定証明書交付申請書（COE申請）

### 概要

| 項目 | 内容 |
|------|------|
| 根拠 | 入管法第7条の2 |
| 様式 | 別記第6号の3 |
| シート構成 | 5シート |
| 提出先 | 地方出入国在留管理局 |
| 提出時期 | 入学予定の6か月前頃 |

### シート構成と出力フィールド

#### シート1 — 申請人情報（表面）

| フィールド | データ型 | 説明 |
|-----------|---------|------|
| 国籍・地域 | テキスト | 申請人の国籍 |
| 生年月日 | 日付 | 申請人の生年月日 |
| 氏名 | テキスト | パスポート記載の氏名 |
| 性別 | 選択 | 男 / 女 |
| 出生地 | テキスト | 出生した都市名 |
| 配偶者の有無 | 選択 | 有 / 無 |
| 職業 | テキスト | 現在の職業 |
| 本国の住所 | テキスト | 本国における住所 |
| 日本の連絡先 | テキスト | 日本到着後の住所（学校寮等） |
| 電話番号 | テキスト | 日本の連絡先電話番号 |
| パスポート番号 | テキスト | 旅券番号 |
| パスポート有効期限 | 日付 | 旅券の有効期限 |
| 入国目的 | 固定値 | 留学 |
| 入国予定日 | 日付 | 来日予定日 |
| 滞在予定期間 | テキスト | 修業期間に対応（1年〜2年） |
| 過去の入国歴 | テキスト | 過去の日本渡航歴 |
| 犯罪歴の有無 | 選択 | 有 / 無 |

#### シート2 — 申請人情報（裏面）

| フィールド | データ型 | 説明 |
|-----------|---------|------|
| 日本語学習歴 | テキスト | 学習期間・機関名 |
| 最終学歴 | テキスト | 卒業校名・卒業年月 |
| 日本語能力 | テキスト | 試験名・レベル |
| 在日親族 | テキスト | 氏名・続柄・在留資格（該当者のみ） |

#### シート3 — 経費支弁関係

| フィールド | データ型 | 説明 |
|-----------|---------|------|
| 経費支弁方法 | 選択 | 本人負担 / 経費支弁者負担 / 奨学金 等 |
| 経費支弁者氏名 | テキスト | 支弁者の氏名 |
| 経費支弁者住所 | テキスト | 支弁者の住所 |
| 申請人との関係 | テキスト | 父 / 母 / その他 |
| 経費支弁者の職業 | テキスト | 勤務先名・職種 |
| 年収 | 数値 | 支弁者の年収（現地通貨） |
| 送金方法 | テキスト | 銀行送金 等 |

#### シート4 — 所属機関情報（学校側記入）

| フィールド | データ型 | サンプル値 |
|-----------|---------|-----------|
| 機関名 | テキスト | 常南国際学院 |
| 所在地 | テキスト | 茨城県つくば市榎戸433-2 |
| 電話番号 | テキスト | 029-875-8828 |
| 設置者 | テキスト | 株式会社 常南国際教育 |
| 代表者 | テキスト | 代表者氏名 |
| 担当者 | テキスト | 事務担当者名 |
| 入学予定コース | テキスト | 進学2年コース 等 |
| 入学予定日 | 日付 | 入学日 |
| 修業期間 | テキスト | 2年 等 |

#### シート5 — 代理人情報

| フィールド | データ型 | 説明 |
|-----------|---------|------|
| 代理人氏名 | テキスト | 申請取次者の氏名 |
| 代理人住所 | テキスト | 申請取次者の住所 |
| 資格 | テキスト | 申請取次者の資格区分 |

## 在留期間更新許可申請書

### 概要

| 項目 | 内容 |
|------|------|
| 根拠 | 入管法第21条 |
| 様式 | 別記第30号 |
| シート構成 | 3シート（申請人用2枚＋所属機関用1枚） |
| 提出先 | 地方出入国在留管理局 |
| 提出時期 | 在留期間満了の3か月前〜満了日まで |

### 更新申請に特有のフィールド

COE申請と共通するフィールドに加え、以下が更新申請特有の項目です。

| フィールド | データ型 | 説明 |
|-----------|---------|------|
| 在留カード番号 | テキスト | 現在の在留カード番号 |
| 現在の在留資格 | 固定値 | 留学 |
| 現在の在留期間 | テキスト | 2年 / 1年3月 / 1年 / 6月 |
| 在留期間の満了日 | 日付 | 現在の在留期間の満了日 |
| 更新の理由 | テキスト | 引き続き在学するため 等 |
| 希望する在留期間 | テキスト | 更新後に希望する期間 |
| 資格外活動の有無 | 選択 | 有 / 無 |
| 資格外活動先名称 | テキスト | アルバイト先の名称 |
| 資格外活動の内容 | テキスト | 業務内容 |
| 週当たり稼働時間 | 数値 | アルバイトの週間時間 |

### 所属機関作成シートの出力フィールド

| フィールド | データ型 | 説明 |
|-----------|---------|------|
| 学校名・所在地 | テキスト | 常南国際学院の情報 |
| 申請人の所属コース | テキスト | 現在在籍するコース |
| 在籍状況 | テキスト | 良好 / 注意 等 |
| 出席率 | 数値 | 直近の出席率 |
| 成績 | テキスト | 成績の概要 |
| 学校の所見 | テキスト | 更新に際しての学校側所見 |

::: warning 出席率と更新の関係
出席率80%以上であれば問題なし。70〜80%未満は理由書が必要。70%未満は原則更新不許可です。
:::

::: tip RINGUALの出力
RINGUALでは、学生マスター・在留情報・出席データ・資格外活動情報から更新申請書の3シートを一括生成します。出席率は自動計算され、所属機関作成シートに反映されます。
:::
