# 業務フロー — 入管報告・届出

## 都度発生する報告・届出

学生の退学・出席不良・受入れの開始終了など、イベント発生時に速やかに届出が必要なものです。

| 報告・届出 | タイミング | 根拠 |
|-----------|-----------|------|
| 退学者の報告 | 退学月の翌月末まで | 告示基準第38号 |
| 出席率5割未満の報告 | 該当月の翌月末まで | 告示基準第39号 |
| 受入れ開始・終了の届出 | 開始/終了から14日以内 | 入管法第19条の17 |
| 各種変更届出 | 変更前または変更後速やかに | 告示基準第42号 |

## 定期的な報告・届出

年間スケジュールに基づき、決まった時期に提出する報告・届出です。

| 報告・届出 | 期限 | 根拠 |
|-----------|------|------|
| 在籍者数に関する届出 | 毎年5月1日および11月1日 | 告示基準第40号 |
| 定期報告（生活指導の概要） | 毎年6月末まで | 告示基準第41号 |
| 運営状況に関する報告書 | 毎年度6月末まで | 日本語教育機関認定法 施行規則第28条 |
| 所在地以外の場所での教育活動届出 | 毎年度4月末まで | 日本語教育機関認定法 施行規則第30条 |
| 受入れ状況に関する届出 | 毎年7月（前年7月〜当年6月分） | 入管法第19条の17（中長期在留者関連） |
| 適正校申請（任意） | 入管の指定時期に申請 | 入管運用基準 |

::: warning 6月末は報告集中月
6月末には「定期報告（告示基準第41号）」「運営状況に関する報告書（施行規則第28条）」が同時に期限を迎えます。準備は5月から開始してください。
:::

## 変更届出（告示基準第42号）

以下の事項に変更が生じた場合、変更前または変更後速やかに届出が必要です。

| No. | 変更事項 |
|-----|---------|
| 1 | 設置者の名称または住所 |
| 2 | 設置者が法人の場合、代表者の氏名 |
| 3 | 設置者が法人の場合、役員の氏名 |
| 4 | 校長の氏名 |
| 5 | 教員の氏名及び担当科目 |
| 6 | 事務局の組織（名称・人員） |
| 7 | 日本語教育機関の名称 |
| 8 | 日本語教育機関の位置（住所） |
| 9 | 収容定員 |
| 10 | 教育課程（カリキュラム） |
| 11 | 入学時期（4月入学、10月入学など） |
| 12 | 修業期間（1年、1年6か月、2年） |
| 13 | 授業時間数 |
| 14 | 授業科目 |
| 15 | 校舎・教室・その他施設 |
| 16 | 学則 |
| 17 | 校舎等の図面 |

## 報告を怠った場合のリスク

1. **適正校認定の取消し**: 報告義務違反は適正校認定の審査でマイナス評価となり、最悪の場合は適正校認定が取り消される
2. **在留資格審査の厳格化**: 適正校でなくなると、学生の在留資格認定証明書（COE）交付審査が厳格化され、必要書類が大幅に増加する
3. **告示抹消**: 重大な違反や改善が見られない場合、日本語教育機関としての告示自体が抹消される可能性がある

## 電子届出システム

入国管理局への届出は、一部を除き**電子届出システム**を利用して行うことができます。

- 受入れ開始・終了の届出（入管法第19条の17）は電子届出に対応
- 定期報告の一部も電子提出が可能
- 変更届出の多くは書面での提出が必要な場合があるため、事前に管轄入管に確認すること
