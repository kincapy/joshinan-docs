# 帳票サンプル — 届出

RINGUALから出力される届出帳票の構造とサンプルです。個人情報はダミーに置換しています。

- [業務フロー](./01-work-flow.md) — 届出のタイミング・方法
- [データ](./02-data.md) — 入管法第19条の17の詳細

---

## 中長期在留者の受入れに関する届出（参考様式2の2）

### 概要

| 項目 | 内容 |
|------|------|
| 根拠 | 入管法第19条の17 |
| 届出者 | 受入れ機関（学校） |
| 届出先 | 地方出入国在留管理局 |
| 届出期限 | 受入れ開始から14日以内 |
| 届出方法 | 電子届出システム / 窓口 / 郵送 |
| 法的性質 | 努力義務（罰則なし、ただし告示基準で事実上の義務） |

### フィールド構成

| フィールド | データ型 | 説明 |
|-----------|---------|------|
| 届出機関名 | テキスト | 常南国際学院 |
| 届出機関所在地 | テキスト | 茨城県つくば市榎戸433-2 |
| 届出機関電話番号 | テキスト | 029-875-8828 |
| 届出年月日 | 日付 | 届出書の提出日 |
| 届出区分 | 固定値 | 受入れの開始 |
| 受入れ対象者氏名 | テキスト | 中長期在留者の氏名 |
| 生年月日 | 日付 | 受入れ対象者の生年月日 |
| 性別 | 選択 | 男 / 女 |
| 国籍・地域 | テキスト | 国籍・地域名 |
| 住居地 | テキスト | 日本国内の住居地 |
| 在留カード番号 | テキスト | 在留カード番号 |
| 在留資格 | 固定値 | 留学 |
| 受入れ開始年月日 | 日付 | 入学日等 |

## 活動機関に関する届出・移籍（参考様式1の3）

### 概要

| 項目 | 内容 |
|------|------|
| 根拠 | 入管法第19条の16 |
| 届出者 | 中長期在留者本人 |
| 届出先 | 地方出入国在留管理局 |
| 届出期限 | 移籍から14日以内 |
| 届出方法 | 窓口 / 郵送 / 電子届出システム |
| 法的性質 | 義務（届出をしない場合は罰則あり） |

::: warning 届出者は学生本人
この届出は学生本人が行うものです。学校は届出を案内・支援する立場です。RINGUALからは学校が代行作成できるよう出力されます。
:::

### フィールド構成

| フィールド | データ型 | 説明 |
|-----------|---------|------|
| 届出者氏名 | テキスト | 中長期在留者本人の氏名 |
| 生年月日 | 日付 | 届出者の生年月日 |
| 性別 | 選択 | 男 / 女 |
| 国籍・地域 | テキスト | 国籍・地域名 |
| 住居地 | テキスト | 日本国内の住居地 |
| 在留カード番号 | テキスト | 在留カード番号 |
| 届出の事由 | 固定値 | 移籍（新たな機関での活動開始） |
| 新たな活動機関名 | テキスト | 転入先の学校名 |
| 新たな活動機関所在地 | テキスト | 転入先の住所 |
| 活動開始年月日 | 日付 | 転入日 |
