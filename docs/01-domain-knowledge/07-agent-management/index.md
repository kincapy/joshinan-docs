# エージェント管理

海外の送出機関・仲介業者（エージェント）の管理を行うカテゴリです。

## 概要

学生の募集は主に海外エージェント経由で行われます。エージェントとの費用精算、請求書管理が主な業務です。

## エージェントとは

エージェント（現地教育機関）とは、現地で留学生を集客し日本語学校に紹介する機関のこと。

- エージェントがいなければ留学生を集客できない
- 悪いエージェントと付き合うと、適当な理由で来る留学生ばかりになり学校運営の負担が増える
- **良いエージェントとの関係構築が日本語学校運営の成否を左右する**

### エージェントのビジネスモデル

| 収入源 | 金額目安 |
|--------|----------|
| 本人からの手数料 | 20〜40万円程度（国による） |
| 日本語学校からの紹介料 | 10〜30万円程度（国・エージェントによる） |

::: info 補足
海外では紹介してもらった人に対してお金を払うのが当たり前の商習慣であり、これ自体を否定するものではない。法外な金額でなければ、日本側が口を出す領域ではない。
:::

### エージェントの種類

エージェントには免許が存在しない（国によっては存在する）ため、種類は多岐にわたる。

| 種類 | 特徴 |
|------|------|
| 自社学校を運営するエージェント | 学生の顔が見える状態で対応。ベスト |
| 紹介のみのエージェント（ブローカー） | 自社学校なし。送り出すだけの場合もある |
| 個人エージェント | 日本在住の外国人や在籍留学生の知り合い経由 |

### 紹介手数料の目安

| 国 | 紹介手数料（1人あたり） |
|----|----------------------|
| ネパール・スリランカ | 10〜15万円 |
| ベトナム | 20万円 |
| 中国 | 30万円 |

::: warning 注意
明確な相場はなく、エージェントが他校からどの程度もらっているかに大きく依存する。安さより「質の良い学生を安定的に送り出してくれるか」を重視すべき。
:::

## エージェントの主な業務

1. 現地での留学生の集客
2. 日本語教育の実施（質にばらつきあり）
3. 入管申請に必要な膨大な書類の準備
4. 学校の情報を留学生に正確に伝達
5. 入国後に留学生が問題を起こした場合のフォロー

## 良いエージェントの見極め方

### エージェントに求める要素

| 要素 | 詳細 |
|------|------|
| **良い学生の紹介** | 日本語教育を一定実施していること |
| **情報の正確な伝達** | 学校の情報を正確に留学生に伝えていること |
| **トラブル対応力** | 入国後の問題に協力して対応できること |
| **書類作成能力** | 訳文を最初の書類と一緒に作成、入管提出順に並べて送付できるエージェントは非常に優秀 |

### 書類の理想的な送付順

1. 申請書
2. 提出書類一覧表及び各種確認書
3. 旅券の写し
4. 最終学校の卒業証明書
5. 日本語能力立証資料
6. 経費支弁書
7. 経費支弁者と申請人との関係を立証する文書
8. 預貯金残高証明書
9. 過去1年間の資金形成過程を明らかにする資料
10. 経費支弁者の職業を立証する資料
11. 過去1年間の経費支弁者の収入を立証する資料
12. 奨学金の給付に関する文書
13. 各種理由書及び裏付け資料
14. 申請人の職業を立証する資料
15. その他、任意で提出する資料

### 最重要テーマ

> 日本語学校の運営上の大きなテーマとして、「入国前の留学生が知っている情報と入国後に知る情報」とのギャップをいかに無くすかが重要。それを一緒に協力して埋めてくれるエージェントと付き合うことが何よりも重要。

## 現地教育機関の構造的問題

### 日本語を教えないエージェントが存在する理由

ビザの交付率が不安定なことが根本原因:

1. 交付率が悪い時期に学生に日本語を教えても、ビザが交付されない場合がある
2. ビザが出なかった場合、「なぜビザが出なかったのか」と追及されるのは現地教育機関側
3. したがって、交付率が悪い時期に日本語を教えるインセンティブが発生しにくい
4. 結果として、ビザが出た際に日本語を十分に学ばないまま留学してくる学生が生まれる

::: tip 現在の状況（2025年）
どの国もビザの交付率が高くなってきている。それでもレベルの低い学生を紹介してくるエージェントは、間違いなく付き合う価値のないエージェント。
:::

## 国別の概要

### 中国

- **別表掲載国**のため、ビザの交付率が非常に高く、募集経路として非常に重要
- 日本の名門大学を志望する留学生が多く、都内の日本語学校に多く在籍
- 地方の日本語学校にはあまり在籍していない
- 中国の失業率の状況から、中流以下の層が就職目的で日本に留学してくる可能性あり
- 中国人にとって日本への留学が非常にカジュアルになっている（ひらがな・カタカナ未習得で来日も）
- 中国人オーナーが運営する日本語学校も増加

### 韓国

- 地方ではほぼ見ることはない
- 韓国系オーナーの日本語学校くらい

### ベトナム

- 技能実習生以外にも留学生として一定数が来ている
- 素行が悪い学生が増えている印象
- 「東京に行きたい」欲求が非常に強く、地方校では在籍管理が厳しくなる傾向

### インドネシア

- 技能実習生として多く来ている国。留学生としては珍しい
- 文法が日本語と近く、日本語力が高い
- 素行が悪い子が少なく、国としては悪くない
- 実習生の増加と経済成長により、今後は留学で来る層も増加見込み

### ミャンマー

- 最近かなり増加。軍事政権の影響で国外に出たい人が多い
- 大学を途中で中退させられた人が多く、基礎学力が高い
- 日本語レベルが非常に高い（入国前にN3まで仕上がる唯一の国）
- 詰め込み型教育が多く、JLPTスコアの割に会話力が低い傾向
- **リスク**: 来日後に特定活動ビザ（難民ビザ）に切り替えるケースもある

### ネパール・スリランカ・バングラデシュ

- 「留学」の主力国。毎日1回以上「留学生を紹介したい」メールが届く
- これらの国の学生が多い学校 = あまり募集を真面目にやっていない可能性
- 留学生を大量に送り込んでいるため、書類作成知識は一定レベル
- 良いエージェントの見極めが非常に困難
- **位置づけ**: それ以外の国でうまくいかなかったときの最終手段として扱うべき
