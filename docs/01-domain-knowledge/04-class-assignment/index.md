# クラス編成

学生 → クラス → 教室 → 教員の割当を管理するカテゴリです。

## クラス体系

常南国際学院では、午前・午後の2部制でクラスを運営しています。

### 2025年度のクラス一覧

| クラス | 時間帯 | JLPTレベル | 備考 |
|--------|--------|-----------|------|
| A1クラス | 午前 | N2 | 上級 |
| A2クラス | 午前 | N3 | 中級 |
| A3クラス | 午前 | N4 | 初中級 |
| A4クラス | 午前 | N5 | 初級 |
| P1クラス | 午後 | — | — |
| P2クラス | 午後 | — | — |
| P3クラス | 午後 | — | — |

- **午前クラス（A系）**: A1〜A4の4クラス。JLPTレベルで分類
- **午後クラス（P系）**: P1〜P3の3クラス

### クラスの設定項目

RINGUALでクラスを登録する際の設定項目：

| フィールド | 内容 | 例 |
|-----------|------|-----|
| クラス名 | 管理用の名称 | A1クラス(2025年度） |
| 印刷用の名称 | 証明書等に印刷される名前 | — |
| レベル | JLPTレベル | N2, N3, N4, N5 |
| 有効期間（開始/終了） | クラスの開講期間 | — |
| サブクラスフラグ | 一時的な所属クラス | 下記参照 |
| 参照レベル | CEFR対応レベル | A1, A2, B1, B2, C1, C2 |

### サブクラス

サブクラスは、学生が**一時的に所属**するクラスです。通常クラスとは別に設定でき、補習や特別授業の管理に使用します。

- 通常クラスとサブクラスの2つの所属を同時に持てる
- フィルターで「サブクラスなし / サブクラスのみ」の切り替えが可能

### CEFRレベルとの対応

RINGUALでは「参照レベル」としてCEFR（ヨーロッパ言語共通参照枠）のレベルも設定可能：

| CEFRレベル | 概要 | JLPT対応（目安） |
|-----------|------|----------------|
| A1 | 基礎段階の初学者 | N5 |
| A2 | 基礎段階 | N4 |
| B1 | 自立した使用者 | N3 |
| B2 | 自立した使用者（上位） | N2 |
| C1 | 熟達した使用者 | N1 |
| C2 | 母語話者レベル | — |

## 1クラスの上限

| 項目 | 値 | 根拠 |
|------|-----|------|
| 1クラスの最大人数 | 20名 | 告示基準 |

## 管理画面のフィルター

RINGUALのクラス一覧画面では、以下のフィルターで絞り込みが可能：

| フィルター | 選択肢 |
|-----------|--------|
| 表示条件 | 開講中 / 開講予定 / 期間内全て |
| 並び順 | クラス名 / レベル / 開始日（昇順/降順） |
| クラスタイプ | すべて / サブクラスなし / サブクラスのみ |
