# クラス編成 データ定義

## クラス (Class)

クラスの基本情報を管理するデータです。午前・午後の時間帯区分や、JLPTレベル、サブクラスの識別を含みます。

| フィールド | 内容 | 例 |
|-----------|------|-----|
| クラス名 | 管理用の名称 | A1クラス(2025年度) |
| 印刷用名称 | 証明書等に印刷される名前 | A1 |
| JLPTレベル | 日本語能力試験のレベル | N2, N3, N4, N5 |
| CEFRレベル | ヨーロッパ言語共通参照枠 | A1, A2, B1, B2, C1, C2 |
| 時間帯区分 | 午前 / 午後 | 午前 |
| サブクラスフラグ | 一時的なクラスかどうか | true / false |
| 最大人数 | 1クラスの上限人数 | 20（告示基準） |
| 年度 | 対象年度 | 2025 |
| 有効期間（開始） | クラスの開講日 | 2025-04-01 |
| 有効期間（終了） | クラスの終了日 | 2026-03-31 |

### クラスの識別パターン

| 時間帯 | クラス名 | JLPTレベル |
|--------|---------|-----------|
| 午前（A系） | A1〜A4 | N2, N3, N4, N5 |
| 午後（P系） | P1〜P3 | — |

## クラス在籍 (ClassEnrollment)

学生がどのクラスに所属しているかを管理するデータです。通常クラスとサブクラスの2種類の在籍タイプがあり、学生は両方を同時に持てます。

| フィールド | 内容 | 例 |
|-----------|------|-----|
| 学生ID | 対象の学生 | — |
| クラスID | 所属するクラス | — |
| 在籍開始日 | クラスへの所属開始日 | 2025-04-01 |
| 在籍終了日 | クラスからの離脱日 | 2026-03-31（在籍中は空） |
| 在籍タイプ | 通常 / サブクラス | 通常 |

### 在籍タイプの使い分け

| 在籍タイプ | 用途 | 備考 |
|-----------|------|------|
| 通常 | 正規のクラス所属 | 1学生あたり1つ |
| サブクラス | 補習・特別授業への一時的な所属 | 通常クラスと並行して所属可能 |

::: info 同一クラスへの重複在籍
同じ学生が同じクラスに、同じ在籍タイプで重複して在籍することはできません。ただし、通常とサブクラスで異なるタイプであれば、同じクラスに所属することは可能です。
:::
